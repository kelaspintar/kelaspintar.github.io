Nilai rata-rata ulangan murid perempuan ialah 81,5.

Rata rata atau mean dari suatu data dapat dicari dengan menggunakan rumus :

[tex]^-x = \frac{{\Sigma}x_i}{n}[/tex]

Keterangan:

[tex]^-x[/tex] = rata-rata

[tex]{\Sigma}x_i}[/tex] = jumlah data

n = banyaknya data

Penjelasan dengan langkah-langkah:

Diketahui:

Murid laki-laki = 12

Murid perempuan = 16

Rata-rata nilai ulangan Matematikanya di kelas tersebut ialah 80.

Ditanya:

Sebelum remedial, nilai rata-rata murid laki-laki di kelas tersebut ialah 78, maka nilai rata-rata ulangan murid perempuan ialah....

Jawab:

Misalkan:

Nilai murid laki-laki = a

Nilai murid perempuan = b

Rata-rata nilai ulangan Matematikanya di kelas tersebut ialah 80.

[tex]\frac{12 \times a + 16 \times b}{12+16} =80[/tex]

12 x 78 + 16 x b = 80 x (12 + 16)

936 + 16b = 2.240

16 b = 2240 - 936

16b = 1304

b = 326/4

b = 81,5

Jadi, nilai rata-rata ulangan murid perempuan ialah 81,5.
