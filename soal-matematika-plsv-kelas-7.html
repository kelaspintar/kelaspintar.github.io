Persamaan linear satu variabel ( PLSV ) adalah sebuah materi di dalam ilmu matematika yang membahas adanya kalimat yang tidak dapat di temukan kebenarannya, dapat di katakan benar dan dapat di katakan salah.

Jadi, Persamaan tersebut identikkan dengan lambang sama dengan (=) dan mengandung satu variabel atau yang dapat mengandung satu variabel dengan pangkat satu.

♡ Berikut Contoh Soal PLSV :1. Diketahui nilai y yang memenuhi persamaan 10y + 50 = 3

Diketahui :

10y + 50 = 3

Ditanya :

Nilai Yang Memenuhi Adalah?

Jawab :

10y + 50 = 3

10y = 3 - 50 ( Pindah Ruas )

10y = -47

y = [tex]\tt -\frac{47}{10}[/tex] ( Memenuhi )

2. Diketahui nilai z yang memenuhi persamaan 40z + 5 = 20

Diketahui :

40z + 5 = 20

Ditanya :

Hasil Yang Memenuhi Adalah?

Jawab :

40z + 5 = 20

40z = 20 - 5 ( Pindah Ruas )

40z = 15

z = [tex]\tt \frac{3}{8}[/tex]  ( Memenuhi )

Pembahasan

PLSV adalah singkatan dari persamaan linear satu variabel. Bentuk umumnya adalah ax + b = c, dengan a, b, c bilangan bulat dan a ≠ 0.

Pada umumnya, persamaan tersebut mempunyai tanda ketidaksamaan :

< ( Kurang Dari ) ≥ ( Lebih Dari Sama Dengan ) > ( Lebih Dari ) = ( Sama Dengan ) ≤ ( Kurang Dari Sama Dengan )

Dan mempunyai bentuk variabelnya, yaitu :

x y z A B D

Dan lain sebagainya.

Adapun Bentuk Umum Persamaan Linear Satu Variabel, Yaitu :

ax + b = c

Keterangan :

a≠ 0 = variabel/peubah.

Suku di sebelah kiri tanda sama dengan untuk ruas kiri dan suku di sebelah kanan tanda sama dengan untuk ruas kanan.

Jawabannya seperti ini:

4x + 32 adalah diagonal 1

6x + 6 adalah diagonal 2

Pada persegi panjang, diagonal 1 sama dengan diagonal 2

Maka untuk contoh soal persamaan linear satu variabel beserta pembahasannya menghasilkan:

4x + 32 = 6x + 6

4x – 6x = 6 – 32

-2x = -26

x = -26/-2

x = 13

Setelah diketahui nilai x sama dengan 13 selanjutnya dilakukan substitusi berikut!

6x + 6

6(13) + 6

19 + 6

25

Substitusi cukup pada salah satu persamaan saja karena nilai diagonal 1 pada persegi panjang sama dengan diagonal 2.

Ini artinya, jika diagonal 2 sama dengan 25 maka demikian juga pada diagonal 1.

Penyelesaian contoh soal persamaan linear satu variabel beserta pembahasannya adalah:

Keliling = 94 cm

Rumus keliling sama dengan 2(p+l)

94 = 14x + 10

84 = 14x

14x = 84

x = 84/14

x = 6

Setelah diketahui x sama dengan 6, selanjutnya tinggal substitusikan ke persamaan panjang dan lebarnya.

Untuk menemukan panjang, substitusinya adalah

5x + 2

5(6) + 2

30 + 2

32

Maka panjangnya sama dengan 32

Sementara itu untuk lebarnya:

2x + 3

2(6) + 3

12 + 3

15

Maka lebarnya sama dengan 15

Jadi, contoh soal persamaan linear satu variabel beserta pembahasannya tersebut memberitahukan bahwa panjang sama dengan 32 dan lebar sama dengan 15 cm.
