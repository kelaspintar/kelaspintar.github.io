26. pengertian dan contoh rumus fungsi matematika untuk smk?


Fungsi adalah suatu relasi khusus yang memasangkan setiap elemen2 daerah asal(domain) tepat satu elemen2 daerah kawan (kodomain) .  jadi agar suaturelasi dapat disebut fungsi harus memenuhi hal2 sbb : semua elemen daerah asal harus memiliki pasangan dan dan setiap pasangan yg ada adalah tepat satu.
contoh : diket himpunan A : {1,2,3,4,5} dan B : {3,5,7,9}. manakah dari relasi dibawah ini yg merupan fungsi 
R1 : { (1,3) (2,5) (3,7) (4,9) (5,3) }
R2 : { (1,3) (1,5) (2,7) (3,9) }
R3 : { (1,5) (2,5) (3,5) (4,5) (5,9) }
jawab :
R1, R3 merupakan fungsi
R2 bukan fungsi karena ada elemen A yang tidak mempunyai pasangan
