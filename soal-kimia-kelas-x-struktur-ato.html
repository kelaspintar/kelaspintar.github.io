8. Tolong dong... Jadi 5^-x nya dijadiin 1/5^x? Ato gimana?


[tex] {5}^{x} + {5}^{2 - x} - 26 = 0[/tex]

[tex] {5}^{x} + {5}^{2} . {5}^{ - x} - 26 = 0[/tex]

[tex] {5}^{x} + \frac{ 25 }{ {5}^{x} } - 26 = 0[/tex]

misal

[tex] y = {5}^{x} [/tex]

[tex]y + \frac{25}{y} - 26 = 0[/tex]

[tex] \frac{ {y}^{2} }{y} + \frac{25}{y} - \frac{26y}{y} = 0 [/tex]

[tex] \frac{{y}^{2} - 26y + 25 }{y} = 0[/tex]

y² - 26y + 25 = 0

(y - 1)(y - 25) = 0

y = 1 atau y = 25

[tex] {5}^{x} = 1 \: atau \: {5}^{x} = 5[/tex]

[tex] {5}^{x} = {5}^{0} \: atau \: {5}^{x} = {5}^{2} [/tex]

x = 0 atau x = 2
