3. rumus pola bilangan aritmatika suku ke n?? sekalian contoh soal nya plisss soalnya besok buat ujian PTS......​


Rumus pola bilangan aritmatika ke n :

Un = a + ( n - 1 ) . b

a = suku awal

b = beda ( u2 - u1 )

n = suku ke n

Contoh soal :

1. pada pola bilangan aritmatika 3 , 5 , 7 , 9 , ..... maka suku ke 8 adalah ..

Dik :

a = 3

b = 2

n = 8

Dit :

U8 = ?

Jawab :

Un = a + ( n - 1 ) . b

U8 = 3 + ( 8 - 1 ) . 2

= 3 + 7 . 2

= 3 + 14

= 17

2. suku ke tujuhbelas dari barisan : 98,92,86,80,.... adalah ....

Dik :

a = 98

b = - 6

n = 17

Dit :

U17 = ?

Jawab :

Un = a + ( n - 1 ) . b

= 98 + ( 17 - 1 ) . - 6

= 98 + 16 . - 6

= 98 + - 96

= 2

3. Rumus suku ke n barisan : 22,19,16,13,10,.... adalah ....

Dik :

a = 22

b = - 3

Dit :

rumus suku ke n = ?

Jawab :

Un = a + ( n - 1 ) . b

= 22 + ( n - 1 ) .- 3

= 22 - 3n + 3

= 25 - 3n

6. contoh soal pts matematika kelas 7 semester 1yang bisa jawab aku kasih point lebih​


1. Cindy mempunyai uang sebesar Rp8.000,00 dan Janet mempunyai uang Rp2.000,00 lebihnya dari uang Cindy. Perbandingan uang Cindy dan Janet adalah ….

a. 2 : 8

b. 8 : 2

c. 4 : 5

d. 5 : 4

Jawaban: c

Pembahasan:

Misalkan uang Cindy = C

              uang Janet = J

Maka diperoleh:

C = 8.000

J = C + 2.000 = 8.000 + 2.000 = 10.000

Sehingga,

C : J = 8.000 : 10.000 = 4 : 5

Jadi, perbandingan uang Cindy dan Janet adalah 4 : 5

telah 6 bulan dipakai mobilnya terjual seharga Rp140.000.000. Tentukan taksiran terdekat kerugian Pak Edi ...

Jawaban:

Harga beli = 160.000.000

Harga jual = 140.000.000

Rugi = 160.000.000 - 140.000.000 = 20.000.000

Persentase kerugian = (rugi : harga asal barang) ×100%

Jawaban:

POLA LANTAI SENI TARI

Dalam sebuah tarian (terutama tari kelompok), pola lantai perlu diperhatikan. Ada beberapa macam pola lantai pada tarian, antara lain :

1. Pola lantai vertikal : Pada pola lantai ini, penari membentuk garis vertikal, yaitu garis lurus dari depan ke belakang atau sebaliknya.

2. Pola lantai Horizontal : Pada pola lantai ini, penari berbaris membentuk garis lurus ke samping.

3. Pola lantai diagonal : Pada pola lantai ini, penari berbaris membentuk garis menyudut ke kana atau ke kiri.

4. Pola lantai melingkar : Pada pola lantai ini, penari membentuk garis lingkaran.
