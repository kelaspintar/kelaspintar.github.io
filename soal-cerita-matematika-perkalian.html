Misalkan
Uang Aqil = x
Uang Ari = y

A. Model matematikanya adalah

x + y = 22000 ...... (i)
x + 3y = 42000 ......(ii)

B. Besar uang masing masing

(x + 3y) - (x + y) = 42000 - 22000
x - x + 3y - y = 20000
2y = 20000
y = 10000

x + y = 22000
x + 10000 = 22000
x = 12000

Jadi uang aqil = Rp. 12.000 dan uang Ari = Rp. 10.000

C. Selisih uang Aqil dan Ari
x - y = 12000 - 10000
x - y = 2000

jadi selisih uang aqil dan ari adalah Rp. 2000

25. Jumlah uang Aqil dan Ari Rp22.000. Jika uang Aqil ditambah dengan tiga kali lipat uang Ari sama dengan Rp42.000,00, tentukanlah: A. Model matematika dari soal cerita tersebut B. Besarnya uang masing masing C. Selisih uang Aqil dan uang Ari


misal uang Aqil = x dan uang Ari = y

model matematika :

x + y = 22.000
x + 3y = 42.000

Besarnya uang masing-masing :

(x + 3y) - (x + y) = 42.000 -22.000
x - x + 3y - y = 20.000
2y = 20.000
y = 20.000/2
y = 10.000

x + y = 22.000
x + 10.000 = 22.000
x = 22.000 - 10.000
