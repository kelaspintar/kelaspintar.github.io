contoh soal

Suatu senyawa disusun dari 0,18 gram karbon, 0,03 gram hidrogen, dan 0,08 gram oksigen. Massa molekul relatif senyawa tersebut adalah 58 dengan Ar H = 1, Ar C = 12, dan Ar O = 16.

Tentukan rumus molekul senyawa itu!

Penyelesaian:

Rumus molekul senyawa dimisalkan dengan CiHjOk.

Massa senyawa = massa C + massa H + massa O

Massa senyawa = 0,18 + 0,03 + 0,08

ADVERTISEMENT

Massa senyawa = 0,29 gram

Jumlah atom karbon

Massa C = ((i x Ar C) / Mr) x massa senyawa

0,18 = i x 12 x 0,29 / 58

i = 3

Jumlah atom hidrogen

Massa H = ((j x Ar H) / Mr) x massa senyawa

0,03 = j x 1 x 0,29 / 58

j = 6

Jumlah atom oksigen

Massa O = ((k x Ar O) / Mr) x massa senyawa

0,08 = k x 16 x 0,29 / 58

k = 1

Jadi, rumus molekul senyawa tersebut adalah C3H6O.

18. Bantu jawab soal kimia ini ya , siapa aja yg jago kimia 1.pada elektrolisis larutan cuso4 dengan elektoda platina , reaksi dikatode yang akan terjadi adalah 2.tulis reaksi lengkap elektrolisis larutan pb(no3)2


(1)
CuSO₄ --> Cu²⁺ + SO₄²⁻

masuk ke katoda: Cu²⁺
reaksi di katoda: Cu²⁺ + 2e --> Cu

(2)
Pb(NO₃)₂ --> Pb²⁺ + 2NO₃⁻

masuk ke katoda: Pb²⁺
reaksi di katoda: Pb²⁺ + 2e --> Pb

masuk ke anoda: NO₃²⁻
reaksi di anoda (oksidasi air): 2H₂O --> O₂ + 4H⁺ + 4e

untuk reaksi total, reaksi di katoda dikalikan dua lalu digabungkan dengan reaksi di anoda, didapatkan:
2Pb²⁺ + 2H₂O --> 2Pb + O₂ + 4H⁺
