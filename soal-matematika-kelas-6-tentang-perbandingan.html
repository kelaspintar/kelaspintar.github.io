Jawaban: 9 : 8

Penjelasan dengan langkah-langkah:

Diketahui keliling dua buah lingkaran masing-masing adalah 72 cm dan 64 cm. Perbandingan jari-jari kedua lingkaran tersebut dapat ditentukan dengan menggunakan rumus keliling lingkaran:

```

K = 2πr

```

Keterangan:

* K = keliling lingkaran (cm)

* π = konstanta phi (22/7 atau 3,14)

* r = jari-jari lingkaran (cm)

Perbandingan jari-jari dua buah lingkaran tersebut dapat ditulis sebagai berikut:

```

r1 : r2 = K1 : K2

```

Substitusi nilai K1 = 72 cm dan K2 = 64 cm ke dalam rumus perbandingan jari-jari lingkaran, maka diperoleh:

```

r1 : r2 = 72 cm : 64 cm

```

```

r1 : r2 = 9 : 8

```

Jadi, perbandingan jari-jari kedua lingkaran tersebut adalah **9 : 8**.

a) volume tabung

b) luas alas tabung

c) luas tutup tabung

d) luas selimut tabung

e) luas permukaan tabung

f) luas permukaan tabung jika tutupnya dibuka

Pembahasan

a) volume tabung

V = π r2 t

V = 3,14 x 20 x 20 x 40 = 50 240 cm3

b) luas alas tabung

Alas tabung berbentuk lingkaran hingga alasnya

L = π r2

L = 3,14 x 20 x 20 = 1256 cm2

c) luas tutup tabung

Luas tutup tabung sama dengan luas alas tabungnya.

L = 1256 cm2

d) luas selimut tabung

L = 2 π r t

L = 2 x 3,14 x 20 x 40

L = 5 024 cm2

e) luas permukaan tabung

Luas permukaan tabung = luas selimut + luas alas + luas tutup

L = 5 024 + 1 256 + 1 256 = 7 536 cm2

atau dengan menggunakan rumus langsungnya

L = 2 π r (r + t)

L = 2 x 3,14 x 20 (20 + 40)

L = 12,56 x 60 = 7 536 cm2

f) luas permukaan tabung jika tutupnya dibuka

L = luas selimut + luas alas = 5 024 + 1 256 = 6280 cm2

atau dari luas permukaan dikurangi dengan luas tutup

L = 7 536 − 1 256 = 6 280 cm2
