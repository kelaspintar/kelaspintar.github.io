10. Soal Latihan USBN 2017 / 2018 Mapel: Matematika (Peminatan) Satuan Pendidikan: SMA Tahun: 2017 / 2018 Kurikulum: Kurikulum 2013


Kelas : 11
Mapel : Matematika
Kategori : Trigonometri Lanjut, Limit
Kata Kunci : Sudut rangkap, persamaan trigonometri, Limit x mendekati tak terhingga
Kode : 11.2.3 (Kelas 11 Matematika Bab 3 - Trigonometri Lanjut), 11.2.7 (Kelas 11 Matematika Bab 7 - Limit)

Pembahasan :

1) cos 2x - 3 cos x + 2 = 0
=> (2 cos² x - 1) - 3 cos x + 2 = 0
Misal cos x = a
=> (2a² - 1) - 3a + 2 = 0
=> 2a² - 3a + 1 = 0
=> (2a - 1)(a - 1) = 0
=> a = 1/2 atau a = 1

Untuk a = 1/2
cos x = 1/2
Kuadran 1 : x = 60° = π/3
Kuadran 4 : x = 300° = (5/3) π

Untuk a = 1
cos x = 1
x = 0° = 0
x = 360° = 2π

HP = {0, π/3, (5/3) π, 2π}


2) Rumus yang digunakan :
Lim √(ax² + bx + c) - √(px² + qx + r)
x=>~
Jika a > p => ~
Jika a = p => (b - q)/(2√a)
Jika a < p => -~

Lim √(4x² + 6x - 7) - 2x - 3
x=>~

Lim √(4x² + 6x - 7) - (2x + 3)
x=>~

Lim √(4x² + 6x - 7) - √(2x + 3)²
x=>~

Lim √(4x² + 6x - 7) - √(4x² + 12x + 9)
x=>~

a = 4, b = 6, c = -7, p = 4, q = 12, r = 9
Karena a = p maka bisa menggunakan rumus :

(b - q)/(2√a)
= (6 - 12)/(2√4)
= -6/4
= -3/2
= -1 1/2
= -1,5

32. BIG POINT!Soal Uji Coba Ujian Nasional (UCUN) 1 SMP/MTs Tahun Pelajaran 2017/2018 Paket B. Mata pelajaran : Matematika Ujian Nasional.Perhatikan gambar diatas!Panjang SQ adalah..A. 25 cmB. 16 cmC. 12 cmD. 9 cmTolong dijawab dengan benar dan menggunakan cara yang lengkap. Yang ngasal saya report ya, Trimakasih sebelumnya.​


Jawaban:

C. 12 cm

Penjelasan:

∆ RPS dgn RP = 20 cm PS = 16 cm

RS² = RP² - PS²

= 20² - 16²

= 400 - 256

= 144

RS. = √144

= 12

∆ RSQ dgn RS = 12 cm sdt R = 90°/2 = 45°

pake rumus sin, cos, tangen

sin r = sisi depan/sisi miring

cos r = sisi samping/sisi miring

tan r = sisi depan / sisi samping

saya menggunakan tan r (tan 45°)

tan 45° = x/RS. ( tan 45° = 1 )

tan 45° = x/12

1 = x/12

x = 1 × 12

x = 12

jadi panjang SQ = 12 cm (C)

maaf kalo salah

40. Dalam satu paket soal ujian terdiri dari paket A sebanyak 5 soal dan paket B sebanyak 6 soal.banyak cara menjawab 5 soal,jika 3 soal dari paket A dan 2 soal dari paket B adalah...cara


Menggunakan kombinasi berdasarkan variasi soal adalah sama apabila berbeda urutan.
Berlaku:
n = Soal A x Soal B
n = C(5,3) x C(6,2)
n = 5!/(2!3!) x 6!/(4!2!)
n = 5.4/2 x 6.5/2
n = 10 x 15
n = 150 cara
