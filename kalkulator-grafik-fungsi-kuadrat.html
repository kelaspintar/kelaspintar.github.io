Hahaaa ..., soalnya bagus sekali, sepintas gak masuk akal, tapi ternyata sangat rasional & bisa dihitung ....

35 x 50

= 35 x (35 + 15)

= 35² + (35 x 15)

= 35² + 15 x (15 + 20)

= 35² + 15² + (15 x 20)

= 35² + 15² + 15 x (15 + 5)

= 35² + 15² + 15² + (15 x 5)

= 35² + 15² + 15² + 5 x (5 + 10)

= 35² + 15² + 15² + 5² + (5 x 10)

= 35² + 15² + 15² + 5² + 5 x (5 + 5)

= 35² + 15² + 15² + 5² + 5² + 5²

Jadi 35 x 50 = 35² + 15² + 15² + 5² + 5² + 5²

Toss dulu ....

19. fungsi x pangkat y di kalkulator​


Jawaban:

Fungsi x pangkat y di kalkulator adalah untuk menghitung hasil perpangkatan antara dua angka, di mana x adalah bilangan pokok (base) dan y adalah pangkat (exponent). Dalam operasi ini, bilangan x akan dipangkatkan dengan bilangan y.

Misalnya, jika kita ingin menghitung 2 pangkat 3, kita akan mengetikkan angka 2 sebagai bilangan pokok (base) dan angka 3 sebagai pangkat (exponent), kemudian kalkulator akan menghitung hasilnya dengan mengalikan bilangan 2 sebanyak 3 kali. Dalam hal ini, hasilnya adalah 2 x 2 x 2 = 8.

Jadi, fungsi x pangkat y di kalkulator digunakan untuk menghitung perpangkatan antara dua angka dan menghasilkan nilai hasilnya

37. Objek baru dalam aplikasi kalkulator dan fungsinya


Jawaban:

Objek baru dalam aplikasi kalkulator bisa berupa tombol persen untuk menghitung persentase, tombol akar kuadrat untuk menghitung akar kuadrat, tombol Pi untuk mengambil nilai Pi, tombol histori untuk melihat riwayat perhitungan, dan mode kalkulasi tambahan seperti mode ilmiah atau pemrograman. Objek-objek ini meningkatkan fungsionalitas dan kemudahan pengguna dalam melakukan perhitungan matematika.

Penjelasan:

Objek baru dalam aplikasi kalkulator dapat menjadi tombol atau fitur tambahan yang dapat meningkatkan fungsionalitas dan pengalaman pengguna. Berikut adalah beberapa contoh objek baru dalam aplikasi kalkulator dan fungsinya:

1. Tombol Persen (%): Tombol persen memungkinkan pengguna menghitung persentase dari suatu angka. Dengan menekan tombol persen setelah memasukkan angka, kalkulator akan secara otomatis menghitung persentase dari angka tersebut.

2. Tombol Akar Kuadrat (√): Tombol akar kuadrat memungkinkan pengguna menghitung akar kuadrat dari suatu angka. Pengguna dapat memasukkan angka dan menekan tombol akar kuadrat untuk mendapatkan hasilnya.

3. Tombol Pi (π): Tombol Pi memungkinkan pengguna mengambil nilai konstanta Pi (π) dengan mudah. Dengan menekan tombol Pi, kalkulator akan memasukkan nilai Pi ke dalam perhitungan matematika.

4. Tombol Histori (History): Tombol histori memungkinkan pengguna melihat riwayat perhitungan sebelumnya. Dengan menekan tombol histori, pengguna dapat melihat dan memilih perhitungan sebelumnya yang telah dilakukan.

5. Mode Kalkulasi (Scientific, Programmer, dll.): Objek baru dalam aplikasi kalkulator dapat berupa mode kalkulasi tambahan, seperti mode ilmiah atau mode pemrograman. Mode-mode ini menyediakan fungsi tambahan seperti perhitungan trigonometri, konversi bilangan, atau operasi biner untuk keperluan khusus.
