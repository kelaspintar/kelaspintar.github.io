20. bagaimana cara mengejakan soal operasi hitung pada aljabar ?


pada penjumlahan
*komutatif= a+b=b+a
*asosiatif=a+(b+c)=(a+b)+c
*distributif=ab+ac=a(b+c)=(b+c)a

operasi penjumlahan bentuk aljabar dapat dilaksanakan jika dan hanya jika SUKU SUKUNYA SEJENIS
Agar bisa menjumlahkan dan mengurangkan dalam bentuk aljabar, variabel dan pangkatnya itu harus sama

#Penjumlahan aljabar :
1. 2a + a = 2a + 1a = 3a
2. -b + 3b = 2b
3. a + c ≠ (tidak bisa dijumlahkan karena variabel tidak sama)
4. p² + 2p² = 3p²
5. p² + p³ ≠ (tidak bisa dijumlahkan karena pangkat tidak sama)
6. x² + 2x - 3 dan -5x² - 4x + 2
[kata "dan" diubah menjadi +]
= x² + 2x - 3 + -5x² - 4x + 2
[urutkan dengan yang variabel dan pangkatnya sama]
= x² - 5x² + 2x - 4x - 3 + 2 
= -4x² - 2x + 1
7. 2p - q - 5 dan p + 7q - 10
= 2p - q - 5 + p + 7q - 10
= 2p + p - q + 7q - 5 + 10
= 3p + 6q + 5

#Pengurangan aljabar :
1. 5a - (-3a) = 5a + 3a = 8a
2. 3b - 7b = -4b
3. 2c - a ≠ (tidak bisa dikurang karena jumlah tidak sama)
4. -3p² + p² = -2p²
5. p³ - p ≠ (tidak bisa dikurang karena pangkat tidak sama
6. x² + 2x - 3 dan -5x² - 4x + 2
[kata "dan" diubah menjadi -]
= x² + 2x - 3 - (-5x² - 4x + 2)
= x² + 2x - 3 + 5x² + 4x - 2
= x² + 5x² + 2x + 4x - 3 - 2
= 6x² + 6x - 5
7. 2xy + 4x - 3y dan -3xy + x - 2y
= 2xy + 4x - 3y - (-3xy + x - 2y)
= 2xy + 4x - 3y + 3xy - x + 2y   
= 2xy + 3xy + 4x - x - 3y + 2y
= 5xy + 3x - y
8. -3xy + x - 2y dari 2xy + 4x - 3y
[kata "dari" diubah menjadi - dan dibalekkan]
= 2xy + 4x - 3y - (-3xy + x - 2y)
= 2xy + 4x - 3y + 3xy - x + 2y
[urutkan dengan yang variabel dan pangkatnya sama]
=  2xy + 3xy + 4x - x - 3y + 2x
9. y² - 5x + 4 dari 2y² - 3y + 7
= 2y² - 3y + 7 - (y² - 5y + 4)
= 2y² - 3y + 7 - y² + 5y - 4
= 2y² - y² - 3y + 5y + 7 - 4
= y² + 2y + 3

40. Cara mengejakan soal . Berat 40cm3 emas adalah 250 gram berapa berat 52cm3 emas


40 cm³ → 250 gram
52 cm³ →    x  gram

pakai perbandingan:
x/250 = 52/40
       x = (52/40).250 gram
       x = 325 gram40 / 52 = 250 / x
x = ( 52 × 250 ) / 40
= 325 gram
Jadi, berat 52 cm3 emas adalah 325 gram
