1) D. 3 cm

5) C. 63,59 cm²

Penjelasan dengan langkah-langkah:

1. r = d/2 = 6/2 = 3 cm

2. d = 2r = 2•0,9 = 1,8 cm

3. k = 2πr = 2•22/7•35 = 44•5 = 220 cm

4. k = 2r + 1/4(2πr) = 2r + 1/2πr = 2•14 + 1/2•22/7•14 = 28 + 11•2 = 28 +22 = 50 cm

5. r = d/2 = 9/2 = 4,5

L = πr² = 3,14(4,5²) = 3,14•20,25 = 63,585 cm² = 63,59 cm²
