9. hasil dari operasi perpangkatan 64 ^ 2 + 16^3/4^5 ​


Jawaban:

Untuk menghitung operasi perpangkatan ini, kita akan ikuti langkah-langkah berikut:

1. Hitung \(64^2\):

\[64^2 = 4096\]

2. Hitung \(16^{3/4}\):

Ini adalah akar keempat dari 16, yaitu \(2\), karena \(2^4 = 16\).

3. Hitung \(4^5\):

\[4^5 = 1024\]

4. Sekarang kita akan melakukan operasi yang tersisa:

\[4096 + 2/1024\]

5. Jika kita tambahkan \(4096\) dan \(2/1024\), kita akan mendapatkan hasil akhirnya:

\[4096 + \frac{2}{1024} = 4096 + \frac{1}{512}\]

Jadi, hasil akhir dari operasi perpangkatan ini adalah \(4096 + \frac{1}{512}\).

33. Dapatkan hasil dari operasi perpangkatan berikut ini 64 pangkat 2 + 16 pangkat 3 per 4 pangkat 5 beserta jalannya


[tex]\frac{64^2+16^3}{4^5}[/tex] = 8

Pembahasan

Ingat Sifat-sifat perpangkatan

aᵇ x aⁿ = aᵇ⁺ⁿ

aᵇ : aⁿ = aᵇ⁻ⁿ

( aᵇ)ⁿ = a ᵇ ˣ ⁿ

([tex]\frac{a}{b}[/tex])ⁿ = [tex]\frac{a^n}{b^n}[/tex]

a ⁻ⁿ = [tex]\frac{1}{a^n}[/tex]

(ab)ⁿ = aⁿ x bⁿ

Penyelesaian soal

[tex]\frac{64^2+16^3}{4^5}[/tex]

= [tex]\frac{(2^6)^2+(2^4)^3}{(2^2)^5}[/tex]

= [tex]\frac{2^1^2+2^1^2}{2^1^0}[/tex]

= [tex]\frac{2.2^1^2}{2^1^0}[/tex]

= [tex]\frac{2^1^3}{2^1^0}[/tex]

= 2¹³⁻¹⁰
