Penjelasan dengan langkah-langkah:

Barisan Aritmatika

Diketahui barisan bilangan 9, 11, 13, 15, 17, ...

Soal a: Penetapan Formula/Rumus Suku Ke-n

Dari barisan bilangan 9, 11, 13, 15, 17, ..., dapat kita ambil:

Suku pertama:
a = 9.Beda/selisih antarsuku:
b = 11 – 9 = 13 – 11 = 2.

Formula/rumus suku ke-n untuk barisan tersebut adalah:

[tex]\begin{aligned}U_n&=a+(n-1)b\\&=9+(n-1)\cdot2\\&=9+2n-2\\U_n&=\boxed{7+2n}\end{aligned}[/tex]
_________________

Soal b: Pengujian dengan Induksi Matematika

Langkah Pertama: Basis Induksi

Untuk n = 1, [tex]U_1 = 7 + 2\cdot1 = 9[/tex] merupakan pernyataan yang benar.

Langkah Kedua: Asumsi/Hipotesis

Diandaikan benar bahwa untuk n = k berlaku:
[tex]U_k=7+2k[/tex]

Maka harus dibuktikan benar pula untuk n = k+1, yaitu
[tex]U_{k+1} = U_k+2[/tex]

Di sini kita menggunakan fakta bahwa beda barisan tersebut adalah 2, dan [tex]U_{k+1}[/tex] adalah suku selanjutnya dari [tex]U_k[/tex].

Langkah Ketiga: Pembuktian

[tex]\begin{aligned}U_{k+1}&=7+2(k+1)\\&=7+2k+2\\&=(7+2k)+2\\U_{k+1}&=U_k+2\quad\blacksquare\end{aligned}[/tex]

KESIMPULAN:
Dari ketiga langkah pengujian dengan induksi matematika tersebut, dapat disimpulkan bahwa formula/rumus suku ke-n yang telah diperoleh, yaitu [tex]U_n=7+2n[/tex], terbukti benar untuk semua n bilangan asli.
_________________

Soal c.

Suku ke-130 adalah:

[tex]\begin{aligned}U_{130}&=7+2(130)\\&=7+260\\&=\boxed{\bf267}\end{aligned}[/tex]

18. Tentukan pola pormula dan buktikan dengan induksi matematika dari barisan4+8+12+16


Penjelasan:

Pola dari barisan ini dapat ditulis sebagai 4n, dimana n adalah nomor suku ke-n. Contohnya, suku pertama adalah 4*1 = 4, suku kedua adalah 4*2 = 8, dan seterusnya.

Sekarang, mari kita buktikan rumus ini dengan menggunakan induksi matematika.

Langkah 1: Basis Induksi

Cek untuk suku pertama (n = 1):

4*1 = 4

Hasilnya benar.

Langkah 2: Hipotesis Induksi

Anggap rumus tersebut benar untuk suku ke-n, yaitu 4n.

Langkah 3: Induksi

Kita akan membuktikan rumus tersebut benar untuk suku ke-(n+1).

Dengan menggunakan hipotesis induksi, kita tahu bahwa suku ke-n adalah 4n.

Sekarang kita perlu membuktikan bahwa suku ke-(n+1) sesuai dengan rumus tersebut:

4(n+1) = 4n + 4

Jika kita menjumlahkan 4n (rumus suku ke-n) dengan 4, maka kita mendapatkan:

4n + 4 = suku ke-(n+1)

Dengan demikian, rumus tersebut benar untuk suku ke-(n+1).

Langkah 4: Kesimpulan

Berdasarkan langkah-langkah di atas, kita telah membuktikan bahwa rumus 4n merupakan pola yang benar untuk barisan ini menggunakan induksi matematika.

23. tentukan pola dari barisan bilangan 12, 13, 14, 15, 16, ... kemudian buktikan dengan induksi matematika​


Jawaban:

Tentukan beda barisan tersebut.

b = U2 - U1

b = 15 - 9

b = 6

Tentukan rumus suku ke-n.

Un = U1 + (n - 1)b

Un = 9 + (n - 1)6

Un = 9 + 6n - 6

Un = 6n + 3

Tentukan jumlah n suku pertama.

Sn = n/2 (U1 + Un)

Sn = n/2 (9 + 6n + 3)

Sn = n/2 (6n + 12)

Sn = n(6n + 12)/2

Sn = (6n² + 12n)/2

Sn = 3n² + 6n

Dari perhitungan di atas, dengan induksi matematika, buktikan bahwa 9 + 15 + 21 + ... + (6n + 3) = 3n² + 6n.

(i) Untuk n = 1

U1 = 3n² + 6n

9 = 3 · 1² + 6 · 1

= 3 + 6

= 9 → (benar)

(ii) Diasumsikan benar untuk n = k, maka 9 + 15 + 21 + ... + (6k + 3) = 3k² + 6k

(iii) Dibuktikan benar untuk n = k + 1

→ 9 + 15 + 21 + ... + (6k + 3) + (6(k + 1) + 3) = 3(k + 1)² + 6(k + 1)

→ 9 + 15 + 21 + ... + (6k + 3) + (6k + 9) = 3(k + 1)² + 6(k + 1)

= (k + 1)(3(k + 1) + 6)

= (k + 1)(3k + 9)

= 3k³ + 12k + 9

Bukti:

Ruas kiri = 9 + 15 + 21 + ... + (6k + 3) + (6k + 9)

= (9 + 15 + 21 + ... + (6k + 3)) + (6k + 9)

= (3k² + 6k) + (6k + 9)

= 3k² + 12k + 9 → (terbukti)

= ruas kanan

Dari (i), (ii), dan (iii) terbukti benar bahwa 9 + 15 + 21 + 27 + 33 ... + (6n + 3) = 3n² + 6n.
