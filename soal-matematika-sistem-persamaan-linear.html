Pak hartono hendak mengangkut 75 ton barang dari gudang ke tokonya. Untuk keperluan itu, ia menyewa dua jenis truk yaitu jenis I dengan kapasitas 5 ton dan jenis II dengan kapasitas 3 ton. Dengan cara sewa demikian, ia diharuskan menyewa truk sekurang kurang nya 21 buah.

Pembahasan :

Misal
Truk jenis 1 = x ==> x ≥ 0
Truk jenis 2 = y ==> y ≥ 0

Kapasitas truk jenis 1 = 5 ton
Kapasitas truk jenis 2 = 3 ton
Barang yang diangkut = 75 ton
5x + 3y ≤ 75

Menyewa truk sekurang - kurangnya 21 buah
x + y ≥ 21


Jadi model matematika dari permasalahan di atas adalah :
5x + 3y ≤ 75, x + y ≥ 21, x ≥ 0, y ≥ 0

Penyelesaian sistem persamaan dua variabel membutuhkan setidaknya 2 variabel persamaan dalam bilangan bulat.

Metode substitusi: Caranya dengan mengganti persamaan yang satu dan lainnya untuk mendapatkan variabel bernilai bilangan bulat.

Metode eliminasi: Caranya dengan menghilangkan salah satu variabel dengan pengurangan terhadap persamaan tersebut.

Metode grafik: Caranya dengan menentukan titik potong garis terhadap sumbu x dan sumbu y, kemudian digambarkan dalam bentuk grafik terhadap titik potong, sehingga himpunan penyelesaian dapat diketahui jika perpotongan garis terbentuk.

Metode grafik

A.

y = x + 3

2x + y + 8 = 0

B.

2x - 4y = 6

4x - 2y = -12

Menentukan titik potong sumbu x (y = 0) dan titik potong sumbu y (x = 0) untuk masing-masing persamaan.

y = x + 3 ... (1)

Untuk y = 0          Untuk x = 0

0 = x + 3               y = 0 + 3

x = -3                    y = 3

Titik koordinat adalah {x, y} yaitu {-3, 0} dan {0, 3}

2x + y + 8 = 0

Untuk y = 0          Untuk x = 0

2x + 0 + 8 = 0      2(0) + y + 8 = 0

2x = -8                  y = -8

x = -4

2x = 6                   -4y = 6

x = 3                     y = -1,5

40. Mapel : MATEMATIKAKelas : 8Bab : SPLDVKategori : MEMBUAT MODEL MATEMATIKA DANMENYELESAIKAN MASALAH SEHARI-HARIYANG MELIBATKAN SISTEM PERSAMAANLINEAR DUA VARIABELSoal :Jumlah panjang dan lebar suatu persegipanjang adalah 32 cm, sedangkanluasnya 240 cm2. Tentukana. panjang dan lebarnya;b. kelilingnya;c. panjang diagonal persegi panjang.


Kelas : 8
Mapel : Matematika
Kategori : Bab 4 Sistem Persamaan Linear Dua Variabel
Kata kunci : SPLDV, model matematika

Kode : 8.2.4 [Kelas 8 Matematika Bab 4 SPLDV]

Penjelasan :

Diketahui :
Jumlah panjang dan lebar = 32 cm
Luas persegi panjang = 240 cm²

Ditanya : 
a. panjang dan lebarnya;
b. kelilingnya;
c. panjang diagonal persegi panjang.

Jawab :

a)
misalkan : panjang = x 
                  lebar = y

kita buat model matematikanya dari pernyataan diatas

x + y = 32  ⇔ y = 32 - x   ... pers I
x × y = 240                      ... pers II

subtitusikan pers I ke dalam pers II

x × y = 240
x (32 - x) = 240
32x - x² = 240
-x² + 32x - 240 = 0
x² - 32x + 240 = 0       (difaktorkan)
(x - 20) (x - 12) = 0
x - 20 = 0     atau   x - 12 = 0
     x₁ = 20                   x₂ = 12

untuk menentukan panjang pake nilai x positif yg terbesar

panjang = 20 cm
lebar = 12 cm

b)
Keliling = 2 (p + l)
             = 2 (20 + 12) cm
             = 2 × 32 cm
             = 64 cm

c)  d² = p² + l²
         = 20² + 12²
         = 400 = 144
         = 544
      d = √544
         = 23,3 cm
    Jadi panjang diagona persegi panjang tersebut adalah 23,3 cm
