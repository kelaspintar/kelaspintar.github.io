Pendahuluan:

Bilangan bulat adalah bilangan yang tidak memiliki batas jumlah anggota nya dari bilangan negatif sampai bilangan positif

Operasi Hitung bilangan bulat sama dengan operasi hitung biasa, pada operasi yang terdapat kali (×) dan bagi (÷) tetap didahulukan di operasi hitungkan

Untuk memudahkan untuk kamu mengerjakan soal soal bilangan bulat kamu dapat menggunakan cara ini :

bilangan negatif ( - ) = merupakan hutang kamu

Bilangan positif ( + ) = merupaka pembayaran hutang kamu

Atau kamu dapat mengerti rumus berikut :

Penjumlahan:

(+) + (+) = (+)

(-) + (-) = (-)

(-) + (+) = (+) , jika (+) > (-)

(+) + (-) = (-) , jika (-) > (+)

Pengurangan

(+) - (+) =  (+) / (-)

 bilangan pertama lebih besar  maka hasilnya (+)

 bilangan kedua yang lebih besar maka hasilnya (-)

(-) - (-) =  (+) / (-)

 bilangan pertama lebih besar  maka hasilnya (+)

 bilangan kedua yang lebih besar maka hasilnya (-)

(+) - (-) = (+)

(-) - (+) = (-)

Perkalian

(+) x (+) = (+)

(-) x (-) = (+)

(+) x (-) = (-)

(-) x (+) = (-)

Pembagian

(+) : (+) = (+)

(-) : (-) = (+)

(+) : (-) = (-)

(-) : (+) = (-)
