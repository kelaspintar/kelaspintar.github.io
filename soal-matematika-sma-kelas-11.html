ㅤ

ㅤ

PEMBAHASAN

Berikut beberapa bentuk umum fungsi trigonometri.

y = p sin (ax + α)y = p cos (ax + α)y = p tan (ax + α)

Dimana:

x = Variabel bebas

y = Variabel tak bebas

p = Konstanta

a = Konstanta

α = Sudut dalam derajat atau radian

ㅤ

Langkah-langkah menentukan fungsi trigonometri jika diketahui grafiknya:

1. Pilihlah beberapa titik koordinat.

2. Lihat titik pada x = 0.

Jika saat x = 0 dan y = 0, maka grafik tersebut fungsi sinus.Jika saat x = 0 dan y = p, maka grafik tersebut fungsi cosinus.Jika saat x = 0 dan y = ∞, maka grafik tersebut fungsi tangen.

3. Substitusi koordinatnya ke bentuk umumnya.

ㅤ

ㅤ

Diketahui:

Grafik fungsi trigonometri akan bernilai -2 saat x = 0.

ㅤ

Ditanyakan:

Fungsi dari grafik tersebut adalah …

ㅤ

Jawab:

LANGKAH 1 : Titik Koordinat

[tex]\sf(0,~-2)[/tex]

[tex]\sf(\frac{\pi}{4},~0)[/tex]

ㅤ

LANGKAH 2 : Saat x = 0

Saat x = 0 ternyata y = -2, berarti fungsinya adalah fungsi cosinus dan diperoleh p = -2.

ㅤ

LANGKAH 3 : Substitusi Koordinat

[tex]\sf Untuk~koordinat~(0,~-2)[/tex]

[tex]\begin{array}{rcl}\sf y&=&\sf p~cos~(ax+\alpha)\\\\\sf-2&=&\sf-2~cos~(k.0+\alpha)\\\\\sf1&=&\sf~cos~\alpha\\\\\sf cos~\alpha&=&\sf1\\\\\sf cos~\alpha&=&\sf cos~0\\\\\alpha&=&\sf0\end{array}[/tex]

ㅤ

[tex]\sf Untuk~koordinat~( \frac{\pi}{4},~0)[/tex]

[tex]\begin{array}{rcl}\sf y&=&\sf p~cos~(ax+\alpha)\\\\\sf0&=&\sf-2~cos~(p.\dfrac{\pi}{4}+0)\\\\\sf0&=&\sf~cos~p.\dfrac{\pi}{4}\\\\\sf cos~p.\dfrac{\pi}{4}&=&\sf0\\\\\sf cos~p.\dfrac{\pi}{4}&=&\sf cos~ \dfrac{\pi}{2}\\\\\sf p.\dfrac{\pi}{4}&=&\sf\dfrac{\pi}{2}\\\\\sf p&=&\sf\dfrac{\cancel\pi}{2}\cdot\dfrac{4}{\cancel\pi}\\\\\sf p&=&\sf2\end{array}[/tex]

ㅤ

[tex]\sf Jadi~fungsi~dari~grafik~tersebut~adalah~\boxed{\sf y=-2\:cos\:2x}.[/tex]
