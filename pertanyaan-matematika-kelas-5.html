4. pelajaran matematika kelas 8berikan 5 contoh pertanyaan fungsi dan jawabannya​


Jawab:

1. Jika f adalah fungsi garis di mana f(1) = 3 dan f(2) = 5, tentukan f(-1). Jawaban: -1.

2. Misalkan f adalah suatu fungsi garis sehingga f(a) = a untuk suatu konstanta a, f(2a) = a + 5 dan f(2a + 7) = a + 12. Tentukan nilai a. Jawaban: a = 5. Pembahasan: Misal f(x) = kx + c. Maka f(2a + 7) - f(2a) = (2ak + 7k + c) - (2ak + c) = 7k = (a + 12) - (a + 5) = 7. Maka 7k = 7, k = 1. Dengan cara yang sama, f(2a) - f(a) = 5 = (2a + c) - (a + c) = a, maka a = 5. (Kita tahu bahwa k = 1, sehingga 2ak = 2a)

3. Andi naik taksi dengan tarif awal Rp7.000. Tarif taksi per km adalah Rp3.000. Jika ia membayar Rp28.000, hitunglah panjang perjalanan Andi dengan taksi. Jawaban: 7 km.

4. Harga taksi Armada jika dinaiki oleh Budi sepanjang 3 km adalah Rp15.000 sedangkan jika dinaiki oleh Coki sepanjang 5 km adalah Rp20.000. Jika Coki dan Budi naik taksi bersama-sama di dalam 1 taksi dan menempuh jarak 8 km, berapakah tarif taksi yang harus mereka bayar? Jawaban: Rp27.500. Pembahasan: 3km + Tarif dasar = Rp15.000, sedangkan 5km + Tarif dasar = Rp20.000. Kurangi kanan dengan kiri, sehingga 2km = Rp5.000. 1 km = Rp2.500. Masukkan ke dalam 3km + Tarif dasar = Rp15.000 agar diperoleh Tarif dasar = Rp7.500. Maka 8km + Rp7.500 = 8*2.500 + 7.500 = 27.500

5. Misalkan f adalah fungsi garis di mana f(g(x)) = x. Jika f(x) = 2x + 5 dan g(x) = ax + b, tentukan nilai a dan b. Jawaban: g Pembahasan: Misal f(x) = y. y = 2x + 5 maka y - 5 = 2x sehingga x = y/2 - 5/2 = g(y). Ubah variabel y menjadi x, diperolehlah g(x) = x/2 - 5/2, maka a = 1/2 sedangkan b = -5/2. (Ini meliputi konsep invers fungsi namun tidak dinyatakan secara langsung.)
