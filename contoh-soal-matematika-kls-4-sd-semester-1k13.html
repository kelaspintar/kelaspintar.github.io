Pembahasan:

Pythagoras:

c² = a² + b²

Rumus:

sin B = b/c

cos B = a/c

tan B = b/a

secan B = c/a

cosecan B = c/b

cotangen B = a/b

cari dulu yang belum diketahui c.

c² = a² + b²

c² = (√12)² + 2²

c² = 12 + 4

c² = 16

c = 4

sin B = b/c

= 2/4

= ½

cos B = a/c

= √12 /4

= 2√3 /4

= ½√3

tan B = b/a

= 2/√12

= 2/(2√3)

= 1/√3 × √3 /√3

= ⅓√3

secan B = c/a

= 4/√12

= 4/(2√3)

= 2/√3 × √3 /√3

= ⅔√3

cosecan B = c/b

= 4/2

= 2

= √12 /2

= 2√3 /2

40. pakai caranya soal matematika kelas 4 SD semester 2 tentang Segitiga​


Rumus keliling segitiga

Keliling segitiga =  a + b + c

Jawaban atas pertanyaan dari segitiga A-E

Pertama-tama, anggap

Kolom ke-2 = a Kolom ke-3 = b dan Kolom ke-4 = c

A. Diketahui = a = 16 cm, b = 15 cm, c =17 cm.

Ditanya = keliling =___?

Keliling segitiga = a + b + c

Keliling segitiga = 16 cm + 15 cm +17 cm = 48 cm.

B. Diketahui = b = 42 dm, c = 46 dm dan keliling = 125 dm.

Ditanya = a =___?

Keliling segitiga = a + b + c

125 dm = a + 42 dm + 46 dm

125 dm - 42 dm - 46 dm = a

83 dm - 46 dm = a

a = 37 dm.

C. Diketahui = a = 20 m, c = 25 m, Keliling = 67 m

Ditanya = b =___?

Keliling segitiga = a + b + c

67 m = 20 m + b + 25 m

67 m - 20 m - 25 m = b

47 m - 25 m = b

b = 22 m.

D. Diketahui = a = 23 cm, b = 26 cm, Keliling = 63 cm

Ditanya = c =___?

Keliling segitiga = a + b + c

63 cm = 23 cm + 26 cm + c

63 cm - 23 cm - 26 cm = c

c = 40 cm - 26 cm = 14 cm.

E. Diketahui = a = 29 mm, b = 32 mm, Keliling = 100 mm

Ditanya = c =___?

Keliling segitiga = a + b + c

100 cm = 29 cm + 32 cm + c

c = 100 cm - 29 cm - 32 cm.

c = 71 cm - 32 cm = 39 cm.
