15. Materi Pendapatan Nasional kelas 11 Semester 1.Apakah yang dimaksud dengan "perhitungan ganda" dalam Metode Produksi?


1. Pendekatan Produksi
     Pendekatan produksi (PDB/PGNP) merupakan pendapatan yang berasal dari penggunaan factor-faktor produksi untuk menghasilkan sesuatu. Pendekatan ini memiliki kelemahan yaitu munculnya double counting atau penghitungan ganda. Perhitungan ganda yang dimaksud yaitu nilai produk sebelumnya akan ditambahkan pada produk-produk turunan berikutnya dan digunakan sebagai nilai akhir produk tersebut. Akibatnya nilai produk akhir menjadi lebih tinggi. Salah satu usaha untuk mengurangi dampak dari double counting yaitu dengan menggunakan pendekatan value added atau nilai tambah. Dalam pendekatan ini nilai produk akan dilihat nilai tambahnya pada produk turunan berikutnya sehingga yang Nampak pada nilai barang akhir yaitu jumlah keseluruhan nilai barang akan sama dnegan nilai akhir produk turunan terakhir. Pendekatan produksi bisa dicari dengan Yield = (P1 x Q1)+(P2 x Q2)+ ….(Pn x Qn).

Penjelasan dengan langkah-langkah:

3 + (-1 x (1 + (-1) ) ) x 3

3 + (-1 x (0) ) ) x 3

3 + 0

keterangan!

• utamakan dalam kurung lalu perkalian dan pembagian lalu pertambahan dan pengurangan

Pertanyaan ⤵

3 + ( -1 × ( 1 + ( -1 ) ) ) × 3 = ...

Jawaban ⤵

= 3 + ( -1 × ( 1 + ( -1 ) ) ) × 3

= 3 + ( -1 × ( 1 - 1 ) ) × 3

= 3 + ( -1 × 0 ) × 3

= 3 + 0 × 3

= 3 + 0

= 3

Skala merupakan perbandingan antara nilai satu dengan nilai lainnya dimana kedua nilai tersebut memiliki satu satuan yang sama (dalam cm, dm, km dll). Skala sendiri berfungsi untuk membantu mengilustrasikan gambaran sebenarnya pada ukuran yang lebih kecil tanpa mengubah bentuk dan perbandingan ukuran.  

secara matematis skala dituliskan dengan :  

Skala = Jarak Pada Peta : Ukuran Sebenarnya

Kembali kepada soal, informasi yang didapatkan diantaranya :

Jarak pada peta = 7 cm

Ditanya : Jarak sebenarnya ?

Mencari jarak sebenarnya dapat kita gunakan rumus untuk mencari skala sehingga kita dapatkan jarak sebenarnya.

Skala = Jarak Pada Peta : Ukuran Sebenarnya

1 : 900 = 7 cm : Ukuran sebenarnya

Ukuran sebenarnya = 7 cm x 900/1

                                 = 6.300 cm

Kemudian kita mengubah ukuran sebenarnya ke dalam satuan meter sesuai dengan permintaan soal. Berdasarkan tangga satuan panjang yang terlampir dibawah diketahui bahwa centimeter ke meter naik sebanyak 2 tangga dengan tiap kenaikan satu tangga dibagi dengan 10. Maka kita dapatkan :

Ukuran sebenarnya = 6.300 cm : ( [tex]10^{jumlah kenaikan}[/tex] )

                                 = 6.300 cm : ( [tex]10^{2}[/tex] )

                                 = 6.300 cm : 100

                                 = 63 m

Jarak anatar kelas 7A dengan 9A pada ukuran sebenarnya adalah 63 m.
