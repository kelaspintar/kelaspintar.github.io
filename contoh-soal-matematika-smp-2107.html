jawab
anggap delta H = dH

dH reaksi = [ ( dH C3H8) + ( dH O2 ) ] - [ 3 ( dH CO2 ) + 4( dH H2O ) ]
dH reaksi = [ -104 + 0 ] - [ -1179  - 1032 ]

7. jumlah bilangan kelipatan 7 antara 100 dan 200 adalah.. a .2100 b. 2107 c.2112 d.2296 ..terbaik!cepat.. beserta cara :)


bilangan antara 100 dan 200 = 105, 112, 119, ... , 196

a = 105
b = 7
Un = 196

Un  = a + (n-1) b
196 = 105 + (n-1) 7
196 = 105 + 7n - 7
  7n = 196 - 98
    n = 98/7
    n = 14

Sn = n/2 (a + Un)
S₁₄ = 14/2 (105 + 196)
      = 7 × 301
      = B.2107
