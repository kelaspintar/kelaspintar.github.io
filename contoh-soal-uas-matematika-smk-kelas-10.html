23. tolong di bantu yah, soal matematika SMK kelas XI ​


a. Persamaan lingkaran berpusat (-2,5), melalui (-1,2)

(x + 2)² + (y - 5)² = r²

→ (-1 + 2)² + (2 - 5)² = 10

→ x² + y² + 4x - 10y + 4 + 25 - 10 = 0

→ x² + y² + 4x - 10y + 19 = 0

b. Persamaan lingkaran berpusat (-2,5) menyinggung garis x - 3 = 0 → x = 3

(x + 2)² + (y - 5)² = r²

→ (3 + 2)² + (0 - 5)² = r²

→ r² = 25

→ (x + 2)² + (y - 5)² = 25

→ x² + y² + 4x - 10y + 4 + 25 - 25 = 0

→ x² + y² + 4x - 10y + 4 = 0

37. matematika SMK kelas 10​


Jawaban:

Sistem Pertidaksamaan Linear 2 Variabel

Penjelasan dengan langkah-langkah:

2x+y≤30

x+2y≤24

x≥0

y≥0

Tentukan nilai max dari

f(x)=3x+3y

Gambar terlampir

2x+y≤30

Menentukan sumbu x maka y=0

2x+0≤30

2x≤30

x≤30/2

x≤15

(15,0)

Menentukan sumbu y maka x=0

2(0)+y≤30

0+y≤30

y≤30

(0,30)

x+2y≤24

Menentukan sumbu x maka y=0

x+2(0)≤24

x+0≤24

x≤24

(24,0)

Menentukan sumbu y maka x=0

0+2y≤24

2y≤24

y≤24/2

y≤12

(0,12)

Titik Temu dari

2x+ y≤30 x 1

x+2y≤24 x 2

2x+ y≤30

2x+4y≤48 -

-3y≤-18

y≥-18/-3

y≥6

2x+y≤30

2x+(6)≤30

2x≤30-6

2x≤24

x≤24/2

x≤12

(12,6)

Nilai maximum dari fungsi

f(x)=3x+3y

(0,12)=3(0)+3(12)=0+36=36

(12,6)=3(12)+3(6)=36+18=54 (√)

(15,0)=3(15)+3(0)=45+0=45

Maka nilai maksimumnya adalah 54.
