2. tolong bantu jawab soal matematika barisan dan deret aritmatika ini​


SOAL NOMOR 4.

Pola deret aritmetika = 200.000 + 225.000 + 250.000 + ....

Suku pertama (a) = U1 = 200.000

Beda (b) = U2 - U1 = 225.000 - 200.000 = 25.000

Rumus suku ke-n adalah

Un = a + (n - 1)•bUn = 200.000 + (n - 1)•25.000Un = 200.000 + 25.000n - 25.000Un = 25.000n + 200.000 - 25.000Un = 25.000n + 175.000

Nilai dari suku ke-36 adalah

Un = 25.000n + 175.000U36 = 25.000(36) + 175.000U36 = 900.000 + 175.000U36 = 1.075.000

Sehingga, jumlah besar tabungan Tristan setelah menabung selama 3 tahun (36 bulan) adalah

Sn = (n/2)•(a + Un)S36 = (36/2)•(200.000 + U36)S36 = (18)•(200.000 + 1.075.000)S36 = (18)•(1.275.000)S36 = 22.950.000

Jadi, besar tabungan Tristan setelah menabung selama 3 tahun adalah Rp 22.950.000,00

12. jumlah n suku pertama suatu deret aritmatika di tentukan oleh rumus Sn=3n²-2 tentukan A. bilangan pertama B. beda C.rumus suku ke-n( un) tolong yg bisa matematika tolong dibantu soal nya besok mau di antar​


A. Bilangan Pertama: 1

B. Beda: b = 8

C. Rumus Suku ke-n: Un = 8n - 7

Pembahasan:

Diketahui:

Jumlah n suku pertama deret aritmatika:

Sn = 3n² - 2

Ditanya:

a. Bilangan pertama.

b. Beda.

c. Rumus suku ke-n.

Jawab:

A. Bilangan Pertama

Sn = 3n² - 2

S1 = U1

=> U1 = 3(1)² - 2

=> U1 = 3 - 2

=> U1 = 1

Jadi, suku pertama deret aritmatika tersebut adalah 1.

B. Beda

S2 = U1 + U2

=> S2 = 3(2)² - 2

=> U1 + U2 = 3(4) - 2

=> 1 + U2 = 12 - 2

=> 1 + U2 = 10

=> U2 = 10 - 1

=> U2 = 9

Beda = U2 - U1

=> b = 9 - 1

=> b = 8

Jadi, beda deret aritmatika tersebut adalah 8.

C. Rumus Suku ke-n

Un = a + (n - 1)b

=> Un = 1 + (n - 1)8

=> Un = 1 + 8n - 8

=> Un = 8n - 7

Jadi, rumus suku ke-n deret aritmatika tersebut adalah Un = 8n - 7
