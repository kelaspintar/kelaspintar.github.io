Jawaban:

Penjelasan dengan langkah-langkah:

8L=8000 CC

8L + 5000 CC = 8000 CC + 5000 CC

= 13000 CC

Kalau ada soal seperti ini Jangan lupa samakan satuannya yaww

8 L + 5.000 CC = .. CC

_____

* 1 CC = 1 ML

_ _ _

-› 8 L + 5.000 ML

= ( 8 × 1.000 ) ML + 5.000 ML

= 8.000 ML + 5.000 ML

= 13.000 ML

= 13.000 CC
