Jawaban:

Untuk pengetikan soal matematika sebaiknya menggunakan fasilitas Format Equation (Format Persamaan) atau Formula Math (Formula Matematika).

Penjelasan:

Untuk pengetikan soal matematika, sebaiknya menggunakan fasilitas Format Equation atau Formula Math. Fasilitas ini biasanya terdapat pada aplikasi pengolah kata seperti Microsoft Word atau Google Docs, dan akan membantu untuk menuliskan soal matematika dengan tanda-tanda dan notasi yang benar sesuai dengan standar penulisan matematika. Fasilitas Symbol juga bisa digunakan untuk menambahkan simbol-simbol matematika. Adapun, Fasilitas Formula Statistik biasanya terdapat pada aplikasi khusus untuk mengolah data statistik, dan tidak terlalu berguna untuk menuliskan soal matematika.

9. bagaimana cara mengetik Microsoft Word di hp kemudian di konversi di laptop atau komputer ?​


Jawaban:

Untuk mengetik di Microsoft Word di HP dan kemudian mengkonversi dokumen ke laptop atau komputer, Anda dapat mengikuti langkah-langkah berikut:

1. Pilih Aplikasi Word di HP: Pastikan Anda memiliki aplikasi Microsoft Word yang sesuai untuk perangkat HP Anda. Banyak perangkat Android dan iOS memiliki versi aplikasi Microsoft Word yang dapat diunduh melalui toko aplikasi resmi seperti Google Play Store atau Apple App Store.

2. Buka Aplikasi Microsoft Word: Setelah mengunduh dan menginstal aplikasi Microsoft Word di HP, buka aplikasi tersebut.

3. Buat atau Buka Dokumen: Pilih opsi untuk membuat dokumen baru atau membuka dokumen yang sudah ada.

4. Mulai Mengetik: Ketik teks atau isi dokumen sesuai kebutuhan Anda menggunakan keyboard virtual di HP.

5. Simpan Dokumen: Setelah selesai mengetik, pastikan untuk menyimpan dokumen di perangkat HP Anda. Pilih opsi "Simpan" dan beri nama dokumen sesuai preferensi Anda.

6. Pindahkan Dokumen ke Laptop atau Komputer: Ada beberapa cara untuk mentransfer dokumen dari HP ke laptop atau komputer:

a. Kirim melalui Email: Anda dapat mengirim dokumen ke alamat email Anda sendiri dan membukanya di laptop atau komputer untuk mengunduhnya.

b. Gunakan Layanan Cloud: Jika Anda menggunakan layanan cloud seperti Google Drive, Dropbox, atau OneDrive, Anda dapat mengunggah dokumen ke akun cloud Anda di HP dan mengaksesnya kembali di laptop atau komputer.

c. Koneksikan Melalui Kabel USB: Anda juga dapat menghubungkan HP ke laptop atau komputer menggunakan kabel USB dan memindahkan dokumen secara langsung melalui fitur transfer file.

7. Buka Dokumen di Laptop atau Komputer: Setelah mentransfer dokumen, buka aplikasi Microsoft Word di laptop atau komputer Anda. Pilih opsi untuk membuka dokumen dan cari dokumen yang Anda transfer sebelumnya.

8. Lanjutkan Pengeditan: Setelah berhasil membuka dokumen di Microsoft Word di laptop atau komputer, Anda dapat melanjutkan pengeditan atau pekerjaan lebih lanjut seperti biasa.

Dengan mengikuti langkah-langkah di atas, Anda dapat dengan mudah mengetik di Microsoft Word di HP dan mengkonversi dokumen ke laptop atau komputer untuk melanjutkan pekerjaan Anda secara lancar.

Gejala

Saat Anda mencoba melihat dokumen Microsoft Word di tampilan tata letak pencetakan, dan jika opsi untuk menampilkan balon dinonaktifkan, ada tidak sederhana klik kanan opsi atau tombol toolbar untuk mengaktifkan balon.

Untuk mengaktifkan balon, Anda harus melakukan hal berikut:

Di Microsoft Office Word 2003 atau Microsoft Word 2002, klik kanan TRK di bilah status di bagian bawah dokumen, dan kemudian klik opsi.

Di Microsoft Office Word 2007, klik tab Tinjau , klik Lacak perubahan dalam grup pelacakan , dan kemudian klik

Ubah pilihan pelacakan.

Di Word 2002, klik untuk memilih kotak centang gunakan balon cetak dan tata letak Web di kotak dialog Lacak perubahan , dan kemudian klik OK.

Di Word 2003, pilih

Selalu di kotak dialog Lacak perubahan dalam Daftar menggunakan balon (cetak dan tata letak Web).

Di Word 2007, pilih selalu di kotak dialog Opsi Lacak perubahan dalam Daftar menggunakan balon (cetak dan tata letak Web).

Pemecahan masalah

Untuk mengatasi masalah ini, gunakan salah satu dari metode berikut ini:

Metode 1: Gunakan makro

Membuat makro, dan kemudian tetapkan ke toolbar. Untuk melakukannya, ikuti langkah-langkah berikut:

Mulai Word.

Di Word 2003 atau Word 2002, arahkan ke makro pada menu alat , dan kemudian klik makro.

Di Word 2007, klik tab pengembang , dan kemudian klik

Makro dalam kelompok kode .

Makro dalam daftar, klik template atau dokumen di mana Anda ingin menyimpan makro.

Dalam kotak nama makro, ketik

ToggleBalloons.

Klik buat untuk membuka Penyunting Visual Basic.

With ActiveWindow.View

If .RevisionsMode = wdBalloonRevisions Then

.RevisionsMode = wdInLineRevisions

Else

.RevisionsMode = wdBalloonRevisions

Pada File menu, klik tutup dan kembali ke Microsoft Word.

Pada menu tampilan , arahkan ke Toolbar, dan kemudian klik Ubah.

Pada tab perintah , di kotak kategori , klik makro.

Makro Toggle balon akan ditampilkan di kotak perintah .

Metode 2: Perubahan tampilan dokumen

Ubah tampilan dokumen. Gunakan tampilan normal jika Anda tidak ingin menampilkan balon, dan mencetak tampilan tata letak (atau tampilan tata letak Web) ketika Anda ingin menampilkan balon.

Di Word 2003 atau Word 2002, klik Normal, Tata letak pencetakan, atau Tata letak Web pada menu tampilan .

Di Word 2007, klik tab tampilan , klik Tata letak Web atau

Metode 3: Menampilkan/menyembunyikan Markup

Anda dapat mengaktifkan tampilan markup semua elemen dalam dokumen atau menonaktifkan.

Di Word 2003 atau Word 2002, klik Markup menu tampilan untuk menampilkan atau menyembunyikan semua elemen markup.

Di Word 2007, klik tab Tinjau , klik Tampilkan Markup dalam grup pelacakan , dan kemudian klik untuk memilih atau menghapus semua kotak centang.

Apakah ini merupakan yang pertama kalinya Anda mengalami masalah ini? Jika ya, silahkan coba restart PC Anda sebelum mencoba metode lainnya. Terkadang, me-restart PC atau Explorer.exe (Windows Explorer) dapat menyelesaikan masalah ini. Berikut ini cara untuk me-restart Windows Explorer:

Langkah 1: Gunakan tombol Ctrl + Shift + Esc untuk membuka Task Manager.

Langkah 2: Klik More Details untuk melihat versi lengkap Task Manager.

Langkah 3: Di bawah tab Processes, cari entri Windows Explorer, klik kanan pada entri tersebut, kemudian klik opsi Restart.

Solusi 2

Jika tidak berfungsi, Mungkin cara selanjutnya akan bekerja total.

Jalankan troubleshooter Start Menu

Microsoft sebenarnya telah memberikan troubleshooter untuk Start Menu Windows 10 yang memungkinkan Anda untuk memperbaiki sebagian besar masalah yang umum terjadi pada menu Start. Untuk mendapatkan peranti lunak ini, silahkan kunjungi halaman ini.

Solusi 3

Menggunakan command PowerShell untuk memperbaiki Start Menu

Langkah 1: Jalankan PowerShell sebagai administrator. Berhubung Start menu nya tidak berfungsi, buka drive Windows 10 Anda, kemudian masuklah ke direktori berikut ini:

Windows\System32\WindowsPowerShell\v1.0

Langkah 2: Temukan file powershell.exe, klik kanan pada file tersebut kemudian klik opsi Run as administrator. Klik Yes jika muncul kotak dialog User Account Control.

Langkah 3: Pada layar PowerShell, copy dan paste lah command di bawah ini, kemudian tekan Enter.

Get-AppXPackage -AllUsers | Foreach {Add-AppxPackage -DisableDevelopmentMode -Register “$($_.InstallLocation)\AppXManifest.xml”}

Proses ini akan sedikit memakan waktu.

Langkah 4: Setelah selesai, klik pada tombol Start atau tekan tombol logo Windows untuk membuka Start Menu

Solusi 4

Buat akun pengguna baru

Jika metode-metode di atas tidak dapat mengatasi masalah Anda, cobalah untuk membuat akun pengguna baru lalu menggunakan akun pengguna baru itu untuk login, bukan akun yang sekarag Anda gunakan. Pada beberapa kasus, Start Menu akan berfungsi seperti biasanya pada akun pengguna baru. Jika berhasil, Anda kemudian bisa memilih untuk menghapus akun pengguna yang lama atau menyimpannya.

Solusi 5

Mereset dan Menginstall Ulang Windows 10  

Masih belum bisa saja? Maka metode terakhir yang bisa Anda coba adalah mereset Windows 10, namun tetap menyimpan data Anda. Anda bisa memilih antara menghapus semua data atau mereset sistem operasi tanpa menghapus data pribadi Anda.
