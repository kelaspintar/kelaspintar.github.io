AO = AC = jari2 lingkaran

maka
AOC adalah segitiga sama kaki

misal :
∠OAC = ∠OCA = x

∠AOC = 180° - 2x

∠ABC = 1/2 . ∠AOC
= 1/2(180° - 2x)
= 90° - x

∠BAC = ∠BAO + ∠OAC = 25° + x
∠BCA = ∠BCO + ∠OCA = 30° + x

∠ABC = 180° - ∠BAC - ∠BCA
90° - x = 180° - (25 + x) - (30 + x)
90 - x = 180 - 25 - x - 30 - x
90 - x = 125 - 2x
2x - x = 125 - 90
x = 35°

∠AOC = 180° - 2x
= 180 - 2(35)
= 180 - 70
= 110°

∠ABC
= 1/2 x ∠AOC
= 1/2 x 110°
= 55°

∠ADC
= 180° - ∠ABC
= 180° - 55°
= 125°
