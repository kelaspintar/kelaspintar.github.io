1. besar sudut c berukuran sama dengan besar sudut b ,besar sudut a berukuran 42° lebih kecil dari sudut b ,besar sudut b adalah


Kelas : 8
Mapel : Matematika
Kategori : Bab 8 - Segitiga dan Segi Empat
Kata Kunci : segitiga, sudut
Kode : 8.2.8 [Kelas 8 Matematika Bab 8 - Segitiga dan Segi Empat]

Pembahasan :
Segitiga adalah bangun dibatasi oleh tiga buah sisi dan memiliki tiga buah titik sudut.

Jumlah ketiga sudut pada segitiga adalah 180°.

Mari kita lihat soal tersebut.
Ralat Soal.
Perhatikan gambar pada lampiran. Segitiga tersebut memiliki besar sudut C berukuran sama dengan besar sudut B dan besar sudut A berukuran 42° lebih kecil dari sudut B. Besar sudut B adalah...

Jawab :
Diketahui besar sudut C berukuran sama dengan besar sudut B, sehingga
∠B = ∠C.
Besar sudut A berukuran 42° lebih kecil dari sudut B, sehingga
∠A + 42° = ∠B
⇔ ∠A = ∠B - 42°

∠A + ∠B + ∠C = 180°
⇔ ∠B - 42° + ∠B + ∠B = 180°
⇔ 3∠B = 180° + 42°
⇔ 3∠B = 222°
⇔ ∠B = 74°

∠A = ∠B - 42°
⇔ ∠A = 74° - 42°
⇔ ∠A = 32°

Jadi, besar sudut B dan C adalah 74° dan besar sudut A adalah 32°.

2 sudut yang saling bertolak belakang, besar sudutnya adalah sama. maka sudut a = sudut b

sudut a = sudut b
(3x - 10)° = (2x + 20)°
3x° - 10° = 2x° + 20°
3x° - 2x° = 20° + 10°
x° = 30°

sudut a + sudut b :
= (3x - 10)° + (2x + 20)°
= (3(30) - 10)° + (2(30) + 20)°
= ( 90 - 10 )° + ( 60 + 20 )°
= ( 80 )° + ( 80 )°
= 160°
