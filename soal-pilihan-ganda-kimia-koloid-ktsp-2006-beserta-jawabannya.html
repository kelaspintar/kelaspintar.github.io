g = 10 m/s²... ANGGAP
va = 3 m/s, ha = 0
vb = 5 m/s, hb = 0,1 m ??
Pa - Pb = ___?

Hukum Bernouli
Pa + ½ ρ va² + ρ g ha = Pb + ½ ρ vb² + ρ g hb
Pa - Pb = ½ ρ (vb² - va²) + ρ g (hb - ha)
Pa - Pb = ½ • 10³ • (5²-3²) + 10³ • 10 (0,1-0)
Pa - Pb = 8000 + 1000

Tentukan nilai % (b/v) untuk mengetahui massa gram NaOH dalam 100 mL larutan.

% (b/v) : % (b/b) x massa jenis
: 30% (b/b) x 1,33 g/mL
: 30% (g/g) x 1,33 g/mL
: 39,9% (g/mL)
: 39,9% (b/v)

39,9% (b/v) artinya 39,9 gram NaOH dalam 100 mL larutan.

Langkah 2
Tentukan jumlah Molaritas NaOH dengan menggunakan rumus sebagai berikut :

Mr NaOH : 23 + 16 + 1
: 40 g/mol

M : mol / L
M : (massa/ Mr) / V. lar
M : massa /( Mr x V.lar
: 39,9 g / (40 g/mol . 100 mL . 10^-3 L/mL)
: 9,975 mol / L
: 9,975 M

Cara cepat mengerjakan soal tersebut. Gunakan rumus berikut :

M : (% (b/b) x m. jenis x 10) / Mr
M : (30 x 1,33 x 10) / 40
M : 9,975 mol /L
