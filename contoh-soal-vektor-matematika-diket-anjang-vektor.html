5. diket vektor u=(x² x)tegak lurus dgn vektor v=(1 -4).nilai x adlh


Nilai x yang memenuhi adalah {0,  4). Untuk menyelesaikan soal ini, kita gunakan formula u . v = 0.

Diketahui:

Vektor u = (x²  x) tegak lurus dgn vektor v = (1  -4)

Ditanya:

Nilai x adalah ...

Jawab:

Dua vektor saling tegak lurus, maka hasil kalinya sama dengan 0.

u . v = 0

x².1 + x.(-4) = 0

x² - 4x = 0

Faktorkan untuk mendapatkan nilai x.

x(x - 4) = 0

x = 0 atau x - 4 = 0

                      x = 4

Nilai x yang memenuhi adalah {0,  4).

7. DIKET vektor a= 3i + 3j dan vektor b= 4i+4j TENTUKAN BESAR SUDUT ANTARA VEKTOR a dan b​


Sudut antar vektor a dan b = 0°

Pendahuluan

Panjang vektor = |a|

= √[(i)² + (j)²]

mencari sudut diantara 2 vektor

a x b = |a| . |b| . cos ®

Langkahnya kita cari perkalian vektor terlebih dahulu.

Setelah mencari panjang vektor. Jika semua sudah kita masukkan rumus dan bisa mencari sudut diantara 2 vektor

Penyelesaian Soal

Diketahui :

vektor a= 3i + 3j

dan vektor b= 4i+4j

TENTUKAN BESAR SUDUT ANTARA VEKTOR a dan b

Ditanya :

Tentukan besar sudut antara vektor a dan b

Jawab :

Sudut antar vektor a dan b = 0°

Pembahasan

vektor a= 3i + 3j

vektor b= 4i+4j

a x b =( 3i + 3j )(4i+4j )

= (3x4 + 3 x 4)

=(12 + 12) = 24

|a| = √(3² + 3²)

= √(9 + 9)

= √18

|b| = √(4² + 4²)

= √(16 + 16)

= √32

a x b = |a|.|b| cos ®

24 = √18 x √32 cos ®

24 = √(9 x 2 x 16 x 2) cos ®

24 = 3 x 2 x 4 cos ®

24 = 24 cos ®

cos ® = 24 /24

cos ® = 1

® = 0

Kesimpulan :

Sudut antar vektor a dan b = 0°

23. DIKET VEKTOR a = 3i + 3j dan vektor b= 4i + 4j TENTUKAN BESAR SUDUT ANTARA VEKTOR a dan b​

|b| = √32

= a · b 

= 3 × 4 + 3 × 4 = 24

a · b = |a||b| · cos θ

24 = 3√2 × 4√2 · cos θ

24 = 24 · cos θ

cos θ = 24/24 

cos θ = 1

θ = 0°

26. 1. Diket : vektor c = (10,2) , vektor d = (5,4x)vektor c. d = 90 maka nikai x adalah2. Diket : vektor a = (2,3) , vektor b = (1,4) ,dan c = (3,5)maka vektor a.(b. c) adalah3. Jika vektor c. d = 16 satuan dan sudut yg terbentuk kedua vektor <(c. d) = 60° maka |c| . |d| =Bantu jawab pakai caranyaa!! ​


1. Kita dapat menggunakan rumus perkalian dot product antara dua vektor c dan d untuk mencari nilai x, yaitu:

c . d = ||c|| ||d|| cos θ

Di mana ||c|| adalah magnitudo vektor c dan ||d|| adalah magnitudo vektor d. Karena vektor c = (10, 2) dan vektor d = (5, 4x), maka ||c|| = √(10² + 2²) = √104 dan ||d|| = √(5² + (4x)²) = √(16x² + 25).

Sudut antara vektor c dan d adalah 90 derajat karena perkalian dot product mereka adalah 90. Oleh karena itu, cos θ = 0 dan kita dapat menyelesaikan persamaan tersebut untuk x:

c . d = 90 = ||c|| ||d|| cos θ = √104 √(16x² + 25) * 0

b
