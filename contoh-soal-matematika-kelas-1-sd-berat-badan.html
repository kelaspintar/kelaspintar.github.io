6. rata rata berat badan kelas V SD sukarukun adalah​


Rata - rata berat badan kelas V SD Sukarukun adalah 36 kg.

Penjelasan :

Nilai rata - rata (dilambangkan dengan x̄) pada suatu kelompok adalah suatu bilangan yang mewakili keseluruhan data pada kelompok tersebut yang didapatkan dari jumlah nilai data dibagi dengan banyaknya data tersebut. Biasanya nilai rata - rata digunakan untuk mengevaluasi data atau membandingkan kelompok data yang satu dengan yang lainnya sehingga lebih cepat dan menggambarkan seluruh data namun tidak dapat digunakan untuk menentukan nilai data tertentu di antara sekelompok data. Rata - rata suatu data ditentukan dengan rumus : x̄ = [tex] \frac{Jumlah \: Data}{Banyaknya \: Data} [/tex].

Agar lebih jelas dalam penerapannya, simak pembahasan soal berikut.

PEMBAHASAN :

Diketahui berat badan 23 siswa kelas V SD Sukarukun adalah sebagai berikut :

5 orang memiliki berat badan 34 kg,

5 orang memiliki berat badan 35 kg,

4 orang memiliki berat badan 36 kg,

3 orang memiliki berat badan 37 kg,

6 orang memiliki berat badan 38 kg.

Hitung rata - rata berat badannya.

• Dari data di atas, jumlah siswanya adalah 23. Itu berarti banyaknya data adalah 23.

• Tentukan jumlah datanya.

Jumlah data = (5 × 34 kg) + (5 × 35 kg) + (4 × 36 kg) + (3 × 37 kg) + (6 × 38 kg)

Jumlah Data = 170 kg + 175 kg + 144 kg + 111 kg + 228 kg

Jumlah Data = 828 kg

• Hitung rata - ratanya.

x̄ = [tex] \frac{Jumlah \: Data}{Banyaknya \: Data} [/tex]

x̄ = [tex] \frac{828 \: kg}{23} [/tex]

x̄ = 36 kg

15. contoh soal ulangan matematika kelas 4 sd tentang mengubah pecahan


Ada 3 bagian:

a. 6/25 (pecahan biasa)
harus diubah dulu menjadi bilangan senilai penyebut 100
untuk mengubah pembilangnya menjadi 100 maka 25 harus dikalikan dengan 4 begitu juga dengan 6 harus dikalikan dengan 4, hasil akhirnya :
6/25 = (6x4) (25x4)
= 24/100
Jadi 6/25 = 24%

b. 1 ½
untuk pecahan campuran, terlebih dahulu kita mengubah pecahan campuran menjadi pecahan biasa,
1 ½ = (2x1+1) /2
= 3/2
Setelah itu baru kita ubah menjadi persen
3/2 = (3x50) (2x50)
= 150/100
jadi, 1 ½ adalah 150%

c. 0,48
Untuk bentuk desimal, langkah yang dilakukan yaitu mengubah desimal ke dalam bentuk pecahan kemudian mengubah pecahan dengan penyebut 100 dan terakhir mengubah ke dalam bentuk persen,
0,48 = 0 + 40/10 + 8/100
= 0 + 40/100 + 8/100
= 48/100
= 48 %
jadi, 0,48 sama saja dengan 48 %

♑ Penjelasan ♑

Pecahan

Pecahan biasa adalah pecahan yang terdiri dari pembilang ( bilangan terbagi dan letaknya ada di atas ) dan penyebut ( bilangan pembagi dan letaknya ada di bawah ). Bentuk dari pecahan adalah a/b. Menyederhanakan pecahan biasa dapat dilakukan dengan membagi pembilang dan penyebut pecahan dengan bilangan yang sama sampai tidak bisa dibagi lagi.

Mengubah pecahan menjadi desimal

Cara mengubah pecahan menjadi desimal dengan membagi pembilang dengan penyebut.

♑ Penyelesaian soal ♑

Diketahui: 4[tex]\frac{1}{5}[/tex] - 0,253[tex]\frac{4}{8}[/tex] - 1,28,8 - 3[tex]\frac{1}{2}[/tex]

Ditanya: Hasil dari pecahan tersebut adalah ...

Dijawab:

1. 4[tex]\frac{1}{5}[/tex] - 0,25 = ...

= [4 ([tex]\frac{1}{5}[/tex] dijadikan pecahan desimal)] - 0,25

= (4,0 + 0,2) - 0,25

= 4,2 - 0,25

= 3,95

2. 3[tex]\frac{4}{8}[/tex] - 1,2 = ...

= [3 ([tex]\frac{4}{8}[/tex] dijadikan pecahan desimal)] - 1,2

= (3,0 + 0,5) - 1,2

= 3,5 - 1,2

= 2,3

3. 8,8 - 3[tex]\frac{1}{2}[/tex] = ...

= 8,8 - [3 ([tex]\frac{1}{2}[/tex] dijadikan pecahan desimal)]

= 8,8 - (3,0 + 0,5)

= 8,8 - 3,5

= 5,3

Kesimpulan: Pecahan biasa adalah pecahan yang terdiri dari pembilang ( bilangan terbagi dan letaknya ada di atas ) dan penyebut ( bilangan pembagi dan letaknya ada di bawah ).Cara mengubah pecahan menjadi desimal dengan membagi pembilang dengan penyebut.4[tex]\frac{1}{5}[/tex] - 0,25 = 3,953[tex]\frac{4}{8}[/tex] - 1,2 = 2,38,8 - 3[tex]\frac{1}{2}[/tex] = 5,3

23. contoh soal berpapasan matematika sd


Contoh 1.

Adit akan berangkat dari Mataram menuju Selong dengan jarak 60 km. Sedangkan Fatir dari Selong menuju Mataram. Jika Adit dan Fatir masing-masing mengendarai motor dengan kecepatan 55 km/jam dan 45 km/jam. Pukul berapa mereka berpapasan jika mereka berangkat bersamaan pada pukul 10.00 ?

Penyelesaian.

Karena mereka berpapasan, itu artinya waktu tempuh Adit dari Mataram ke tempat berpapasan sama dengan waktu tempuh Fatir ke tempat berpapasan. Sehingga berakibat

Misal jarak tempuh Adit ke tempat mereka berpapasan adalah  dan jarak tempuh Adit ke tempat mereka berpapasan adalah  serta kecepatan Adit dan Fatir adalah  dan . Berakibat  atau dengan kata lain . Sehingga diperoleh

Jadi, waktu yang dibutuhkan agar Adit dan Fatir berpapasan adalah

 jam atau 36 menit

Dengan kata lain, mereka akan berpapasan pada pukul 10.00 + 36 menit = 10.36.

A. 10 ons

Jawaban: B

Jumlah gula pasir adalah 1 kg 2 ons

1 kg = 10 ons

10 ons + 2 ons = 12 ons

Gula pasir diambil ibu 4 ons.

Sisa gula = jumlah gula pasir di toples - jumlah gula pasir yang diambil ibu

Sisa gula = 12 ons - 4 ons

Sisa gula = 8 ons

35. contoh soal sifat asosiatif dan distributif kelas 5 sd mapel matematika


Kelas : V (5 SD)
Materi : bilangan bulat
Kata Kunci : bilangan, bulat, sifat-sifat, asosiatif, distributif

Pembahasan :
1. Penjumlahan
Bersifat asosiatif, sehingga untuk sembarang bilangan bulat a, b, dan c berlaku :

(a + b) + c = a + (b + c)

contoh :

(2 + 6) + 8 = 2 + (6 + 8)

⇔ 8 + 8 = 2 + 14

⇔ 16 = 16


2. Pengurangan

tidak bersifat asosiatif, sehingga untuk sembarang bilangan bulat a, b, dan c tidak berlaku:

(a – b) – c ≠ a – (b – c).

contoh :

(3 - 5) - 7 ≠ 3 - (5 - 7)

⇔ -2 - 7 ≠ 3 - (-2)

⇔ -9 ≠ 5


3. Perkalian

Bersifat asosiatif, sehingga untuk sembarang bilangan bulat a, b, dan c, berlaku :

(a x b) x c = a x (b x c)

contoh :

(2 x 5) x 6 = 2 x (5 x 6)

⇔ 10 x 6 = 2 x 30

⇔ 60 = 60.


Bersifat distributif, sehingga untuk sembarang bilangan bulat a, b, dan c berlaku :

i. distributif terhadap penjumlahan

a x (b + c) = (a x b) + (a x c)

contoh : 

–5 x [9 + (–1)] = [–5 x 9] + [–5 x (–1)] 

⇔ –5 x 8 = –45 + 5

⇔ –40 = –40


ii. distributif terhadap pengurangan

a x (b – c) = (a x b) – (a x c)

contoh :

7 x [–10 – 12] = [7 x (–10)] – [7 x 12]

⇔ 7 x -22 = -70 - 84

⇔ -154 = -154.


4. Pembagian

Tidak bersifat asosiatif, sehingga Untuk sembarang bilangan bulat a, b, dan c berlaku:

(a : b) : c ≠ a : (b : c)

contoh :

(8 : 4) : 2 ≠ 8 : (4 : 2)

⇔ 2 : 2 ≠ 8 : 2

⇔ 1 ≠ 4


Bersifat distributif, sehingga untuk sembarang bilangan bulat a, b, dan c berlaku :

i. distributif terhadap penjumlahan

(a + b) : c = (a : c) + (b : c)

contoh : 

(12 + 6) : 2 = (12 : 2) + (6 : 2)

⇔ 18 : 2 = 6 + 3

⇔ 9 = 9


ii. distributif terhadap pengurangan

(a – b) : c = (a : c) – (b : c)

contoh :

(21 – 9) : 3 = (21 : 3) – (9 : 3)

⇔ 12 : 3 = 7 - 3

⇔ 4 = 4

 

Semangat!
