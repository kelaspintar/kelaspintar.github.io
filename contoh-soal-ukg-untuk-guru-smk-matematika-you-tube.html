33. apakah bisa saya seorang guru mencari bahan ajar atau soal akuntansi smk?


Penjelasan:

Bisa kak, kakak mendownload bahan ajar akuntansi dari website yang ada yang memuat kurikulum saat ini, rancangan belajarnya, hingga materi yang digunakan. Untuk materi bisa menggunakan e book Financial Accounting karya Weygandt, Kimmel, Kieso dan Intermediate Accounting karya Weygandt, Kieso, Warfield dan buku akuntansi buatan Indonesia.

Namun e book diatas untuk jenjang perguruan tinggi, kakak harus mengikuti kurikulum dan materi sesuai dengan porsi siswa SMK. Lalu yang terpenting di akuntansi ada yang namanya ujian praktek siklus akuntansi. Ujian praktek ini ada dua bagian, pertama mengerjakan siklus akuntansi secara manual dan komputerisasi (menggunakan MYOB, Accurate, Zahir, dsb). Untuk ujian praktek bisa dikoordinasikan dengan pihak sekolah seperti penyediaan lab komputer.

Untuk ujian praktek, alangkah bijaknya dapat dipersiapkan dahulu untuk latihannya, kakak bisa mendownload soal UKK Akuntansi dari website yang ada dan dapat menjadi referensi untuk membekali siswa dalam mempersiapkan ujian/Ujian Praktek.

38. pengertian dan contoh rumus fungsi matematika untuk smk?


Fungsi adalah suatu relasi khusus yang memasangkan setiap elemen2 daerah asal(domain) tepat satu elemen2 daerah kawan (kodomain) .  jadi agar suaturelasi dapat disebut fungsi harus memenuhi hal2 sbb : semua elemen daerah asal harus memiliki pasangan dan dan setiap pasangan yg ada adalah tepat satu.
contoh : diket himpunan A : {1,2,3,4,5} dan B : {3,5,7,9}. manakah dari relasi dibawah ini yg merupan fungsi 
R1 : { (1,3) (2,5) (3,7) (4,9) (5,3) }
R2 : { (1,3) (1,5) (2,7) (3,9) }
R3 : { (1,5) (2,5) (3,5) (4,5) (5,9) }
jawab :
R1, R3 merupakan fungsi
R2 bukan fungsi karena ada elemen A yang tidak mempunyai pasangan

Zulfa mendapat tugas dari gurunya untuk mewarnai sebuahminiatur rumah yang bentuknya merupakan gabungan darisebuah balok dan sebuah prisma segitiga seperti pada gambar.Jika satu tube cat air cukup untuk mewarnai area seluas 16 cmpersegi, maka banyak cat air yang diperlukan adalah ...a. 16 tube b. 96 tube c. 131 tube d. 160 tubetolong dijwab!​


Banyak cat air yang diperlukan adalah C. 131 tube. Caranya akan dijelaskan dipembahasan ya..

Pembahasan

Untuk menghitung berapa banyak cat yang dibutuhkan, ditentukan dahulu luas permukaan dari balok dan prisma segitiga tersebut. Rumus luas permukaan dari balok tersebut yaitu p x l + 2(p x t) + 2(l x t). Kenapa bagian panjang x lebar hanya 1 kali? Karena bagian atas balok yang menyatu dengan prisma segitiga tidak ikut dicat.

Sedangkan rumus luas permukaan dari prisma segitiga adalah 2 x luas segitiga + 2 x luas persegi panjang (Persegi panjang hanya terhitung 2, karena bagian bawah prisma segitiga tidak ikut dicat ya).

Luas permukaan balok = p x l + 2(p x t) + 2(l x t).

= (30 x 16) + 2(30 x 10) + 2(16 x 10)

= 480 + 600 + 320 = 1.400 cm²

Sebelum menentukan luas permukaan prisma segitig , ditentukan dahulu sisi miring pada prisma segitiga dengan rumus pythagoras :

Sisi miring = √(½ x alas)² + tinggi ²

= √(½ x 16)² + 6² = √64 + 36 = √100 = 10

Luas permukaan prisma segitiga = 2(½ x a x t) + 2(p x l)

= 2(½ x 16 x 6) + 2(30 x 10)

= 96 + 600 = 696 cm²

Luas total permukaan bangun tersebut : 1.400 + 696 = 2.096 cm².

Sehingga cat yang dibutuhkan sebanyak : 2.096 / 16 = 131 tube.
