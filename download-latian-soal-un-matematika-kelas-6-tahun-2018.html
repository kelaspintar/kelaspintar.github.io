rumus:

[tex] \alpha + \beta = - \frac{b}{a} [/tex]

[tex] \alpha + \beta = - \frac{ - 2}{1} = 2 [/tex]

[tex] \alpha \times \beta = \frac{c}{a} [/tex]

[tex] \alpha \times \beta = \frac{1}{1} = 1[/tex]

[tex] \frac{1}{\alpha } + \frac{1}{ \beta } [/tex]

[tex] = \frac{2}{1} [/tex]

[tex] = 2[/tex]

40. diagram batang di atas menunjukkan data jumlah siswa kelas 6 SD Melati Tahun 2012-2016 berapa rata-rata jumlah siswa kelas 6 SD Melati setiap tahunnyaTolong di jawab dengan benar​


Jawaban:

Rumus Rata-Rata / Mean ↓

Jumlah Seluruh Data ÷ Banyaknya Data

2012 = 40 murid

2013 = 38 murid

2014 = 37 murid

2015 = 36 murid

2016 = 38 murid

= 40 + 38 + 37 + 36 + 38

= 189

Jumlah Seluruh Data = 189

Banyaknya Data = 5 ( 2012 - 2016 )

189 ÷ 5 = 37,8

Maka Nilai Rata-Rata nya Adalah 37,8
