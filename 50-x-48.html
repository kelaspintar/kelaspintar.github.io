11. Diketahui suatu data: 65, 48, x, 50, ,58, 48, 65, 62 Diketahui rata rata data tersebut 57. Nilai x adalah...


Diketahui Data : 65, 48, x, 50, 58, 48, 65, 62. Rata rata nya adalah 57. Maka nilai x adalah 60.

Penyelesaian Soal :

Diketahui : Data : 65, 48, x, 50, 58, 48, 65, 62

                  X bar = 57

                  n = 8

Ditanya : Nilai x ?

Jawab :

LANGKAH PERTAMA (I)

Hitung nilai total keseluruhan data (∑X) dengan menggunakan cara sebagai berikut :

X bar = ∑X/ n

Keterangan : X bar = Nilai rata rata  

                       ∑X = Jumlah nilai

                      n = banyaknya data

Maka caranya yaitu :

∑X = X bar × n

     = 57 × 8

     = 456

LANGKAH KEDUA (II)

Hitung nilai x dengan menggunakan cara sebagai berikut :

∑X = 65 + 48 + x + 50 + 58 + 48 + 65 + 62

456 = 65 + 48 + x + 50 + 58 + 48 + 65 + 62

456 = 396 + x

x = 456 - 396

x = 60

∴ Kesimpulan nilai x adalah 60.

31. 50 kembangkan cara cepat untuk menghibunglean Perkalian 50. Gunakan cara tersebut untuk menghitung bilangan sekitar a-47 x 49 b. 48 x 46 c. 52 x 53 d-53 x 48..​


Untuk menghitung perkalian bilangan sekitar 50 dengan mengembangkan cara cepet.

a. 47 × 49 =  2303B. 48 × 46 = 2208C. 52 × 53 = 2756 D. 53 × 48 = 2544

Penjelasan dengan langkah-langkah

Diketahui :

a. 47 × 49

B. 48 × 46

C. 52 × 53

D. 53 × 48

Ditanya :

Menghitung perkalian bilangan sekitar 50 dengan mengembangkan cara cepet.

Jawab :

Untuk menghitung perkalian yang bilangannya sekitar 50 kita bisa mengguakan bentuk aljabar yaitu (a - b) (a - c) = a² - a (b + c) + bc.

Menghitung perkalian bilangan sekitar 50

a. 47 × 49 = (50 - 3) (50 - 1)

                 = 50² - 50 (3 + 1) + (3 × 1)

                 = 2500 - 200 + 3

                 = 2303

B. 48 × 46 = (50 - 2) (50 - 4)

                 = 50² - 50 (2 + 4) + (2 × 4)

                 = 2500 - 300 + 8

                 = 2208

C. 52 × 53 = (50 + 2) (50 + 3)

                 = 50² + 50 (2 + 3) + (2 × 3)

                 = 2500 + 250 + 6

                 = 2756

D. 53 × 48 = (50 + 3) (50 - 2)

                 = 50² + 50 (3 - 2) + (3 × -2)

                 = 2500 + 50 - 6

                 = 2544
