13. 1.-28: (10-11) × (-2-3) 2. ( 7 + ²73 ) × ( 3 ²/2 - 1/7 ) = 6/5​


Jawab:

Mari kita selesaikan perhitungan dari dua ekspresi tersebut:

1. Perhitungan untuk ekspresi (1):

  -28 : (10-11) × (-2-3)

Langkah 1: Lakukan operasi dalam tanda kurung terlebih dahulu.

  -28 : (-1) × (-2-3)

Langkah 2: Lakukan perkalian dalam tanda kurung.

  -28 : (-1) × (-5)

Langkah 3: Lakukan pembagian.

  28 × (-5)

Langkah 4: Hasil akhir.

  -140

Jadi, hasil dari (1) adalah -140.

2. Perhitungan untuk ekspresi (2):

  (7 + √73) × (√(3/2) - 1/7) = 6/5

Langkah 1: Hitung akar kuadrat dari 73.

  √73 ≈ 8.544

Langkah 2: Gantikan nilai dari √73 dan √(3/2) ke dalam ekspresi.

  (7 + 8.544) × (√(3/2) - 1/7) = 6/5

Langkah 3: Lakukan penjumlahan di dalam tanda kurung pertama.

  (15.544) × (√(3/2) - 1/7) = 6/5

Langkah 4: Hitung nilai dari √(3/2).

  √(3/2) ≈ 1.225

Langkah 5: Gantikan nilai dari √(3/2) ke dalam ekspresi.

  (15.544) × (1.225 - 1/7) = 6/5

Langkah 6: Lakukan pengurangan di dalam tanda kurung kedua.

  (15.544) × (1.225 - 0.143) = 6/5

Langkah 7: Lakukan perkalian.

  (15.544) × (1.082) = 6/5

Langkah 8: Hitung nilai dari perkalian.

  16.827008 ≈ 6/5

Langkah 9: Persempit hasil menjadi pecahan yang sederhana.

  Hasil akhir: 3/5

Jadi, hasil dari (2) adalah 3/5.


14. uengan1. Sederhanakan:a.381 + 533 - 18b. (73 +8)(2 73-5)​


Pendahuluan.

[tex]\bold{\blue{Bilangan \:Bulat}}[/tex]

Bilangan bulat adalah bilangan yg terdiri dari bilangan positif dan negatif.Operasi bilangan bulat terdiri dari banyak bilangan, antara lain, bilangan cacah, bilangan asli, bilangan prima, bilangan nol, bilangan ganjil dan bilangan genap.

Bilangan bulat dibagi menjadi 3 yaitu bilangan positifif, nol dan negatif.

Bilangan positif : { 1, 2, 3, 4, 5, 6, 7, ..... }

Bilangan nol : { 0 }

Bilangan negatif : { -1, -2, -3, -4, -5, -6, .... }

Dalam operasi bilangan bulat, perkalian diutamakan, lalu baru pembagian, pertambahan, kemudian pengurangan.Namun, jika ada operasi di dalam kurung, maka yg didalam kurung harus diselesaikan terlebih dahulu.
