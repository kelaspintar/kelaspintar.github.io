Diketahui sebuah segitiga dengan panjang sisi a, b, c (a sisi terpendek dan c sisi terpanjang)

Segitiga tersebut termasuk segitiga lancip jika

c² < a² + b²

Segitiga tersebut termasuk segitiga siku-siku jika

c² = a² + b²

Segitiga tersebut termasuk segitiga tumpul jika

c² > a² + b²

==================================

1.

Diketahui:

a = 7

b = 24

c = 25

c² = 25² = 625

a² + b² = 7² + 24² = 49 + 576 = 625

Karena c² = a² + b², maka segitiga ABC dgn panjang sisi 7 cm, 24 cm, 25 cm termasuk segitiga siku-siku.

2.

Diketahui:

a = 9

b = 15

c = 18

c² = 18² = 324

a² + b² = 9² + 15² = 81 + 225 = 306

Karena c² > a² + b², maka segitiga ABC dgn panjang sisi 9 cm, 15 cm, 18 cm termasuk segitiga tumpul.

3.

Diketahui:

a = 8

b = 9

c = 15

c² = 15² = 225

a² + b² = 8² + 9² = 64 + 81 = 145

Karena c² > a² + b², maka segitiga ABC dgn panjang sisi 8 cm, 9 cm, 15 cm termasuk segitiga tumpul.
