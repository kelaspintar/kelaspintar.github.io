33. kak tolongin dong ini sebenarnya udh di upload tapi aku upload ulang karena belum ada cara lengkapnya deadline nya besok makasih​


Jawaban:

PENDAHULUAN

Halo guys, kembali berjumpa lagi bersama saya @elB0YVERScool, Nah, kali ini saya akan membantu kalian semua untuk mengerjakan sejenis soal tentang diagram data. Kalian tahu tidak, Diagram data itu apa, dan berapa banyak macamnya? Kalau belum tahu, mari akan kakak jelaskan, Simak baik baik ya!

[tex]~[/tex]

Untuk membentuk data dalam sebuah diagram, sebelumnya kita memerlukan data tersebut dalam bentuk tabel. Tabel terdiri dari beberapa kotak yg berisi kumpulan data seperti, jumlah/frekuensi/dan totalnya.

[tex]~[/tex]

Diagram ada banyak jenisnya, tapi untuk kelas ini, yg perlu dipelajari cukup 3 dulu, yaitu:

MACAM MACAM DIAGRAM

Diagram Batang

Diagram batang sesuai dengan namanya, diagram tersebut berbentuk batang. Tata letaknya yaitu dengan menempatkan nama objek di samping kiri, dan frekuensi dituliskan di bagian bawah. Jadi kita lambang kan seperti huruf "L". Dimana; l untuk nama objek nya, dan _ untuk banyak data/Frekuensi. Tinggal sesuaikan objeknya dengan berapa banyak frekuensi, lalu gambarlah batangnya sesuai batas frekuensinya.

Diagram Garis

Hampir sama seperti diagram batang, hanya saja bukan menggunakan batang yg berbentuk persegi panjang melainkan menggunakan garis lurus.

Diagram Lingkaran

Diagram Lingkaran bisa menerapkan 2 sistem, yaitu derajat dan persen. Cara mengubah data murni ke bentuk desimal dan persen yaitu:

- Desimal:[tex] \frac{frek}{jumlah \: frek} \times {360}^{o} [/tex]

-

Untuk persen sama dengan desimal, hanya saja, 360° nya diganti dengan 100%

[tex]~[/tex]

PEMBAHASAN

[tex]~[/tex]

DIKETAHUI PADA SOAL:

Data Pekerjaan Orang Tua Siswa Kelas V

[tex]\begin{gathered}\boxed{\begin{array}{c|c}\underline{\bold{Pekerjaan}}&\underline{\bold{Banyak~Siswa}}\\\\\ Pegawai~Negeri&6\\\\Pegawai~Swasta&15\\\\Wiraswasta&9\\\\Total&30\end{array}}\end{gathered} [/tex]

→ Dimana Pekerjaan sebagai nama objeknya, dan Banyak siswa sebagai banyak data / Frekuensinya.

[tex]~[/tex]

DITANYA:Dalam bentuk diagram lingkaran?

[tex]~[/tex]

JAWAB:

Pegawai Negeri:

= [tex]\frac{6}{3\cancel0}×{10\cancel0}[/tex]

= [tex]\frac{6}{3}×{10}[/tex]

= 60/3 = 20%

[tex]~[/tex]

Pegawai Swasta:

= [tex]\frac{15}{3\cancel0}×{10\cancel0}[/tex]

= [tex]\frac{15}{3}×{10}[/tex]

= 150/3 = 50%

[tex]~[/tex]

Wiraswasta:

= [tex]\frac{9}{3\cancel0}×{10\cancel0}[/tex]

= [tex]\frac{9}{3}×{10}[/tex]

= 90/3 = 30%

[tex]~[/tex]

Lalu, tinggal dibuat diagram lingkaran nya sesuai ukuran.

*Untuk ukuran 25% itu = ¼ lingkaran

75% = 3/4 lingkaran

50% = ½ lingkaran

[tex]~[/tex]

Terlampir ya-!
