9. diketahui panjang KL 20cm panjang mn adalah


Panjang MN adalah 5√3 cm.

Pembahasan

Kita akan membahas mengenai trigonometri, terlebih khususnya mengenai aturan sinus. Aturan sinus rumusnya adalah

a / sin A = b / sin B = c / sin C

Sekarang, kita akan mulai menyelesaikan persoalan secara step by step sebagai berikut:

Langkah 1 : Mencari panjang ML

Untuk mencari panjang ML, kita akan menggunakan aturan sinus, sebagai berikut:

ML / Sin K = KL / Sin M

ML / Sin 60° = 20 / Sin 90°

ML / √3/2 = 20 / 1

ML = 10√3 cm

Langkah 2 : Mencari panjang MN

Sebelum mencari panjang MN, kita perlu mengetahui besar sudut L terlebih dahulu.

∠K + ∠L + ∠M = 180°

60° + ∠L + 90° = 180°

∠L = 180° - 60° - 90°

∠L = 30°

Sin ∠L = MN / ML

Sin 30° = MN / 10√3

1/2 = MN / 10√3

MN = 5√3 cm
