Jawab:

Nomor 27 : C

Penjelasan dengan langkah-langkah:

Nomor 27 :

Garis Singgung persekutuan luar = GSPL = 24 cm

Jari-jari Lingkaran kecil = r = 4 cm

Jarak kedua titik pusat = JP = 25 cm

Jari-jari lingkaran besar = R = ..... cm

GSPL² = JP² - (R - r)²

(R - r)² = JP² - GSPL²

(R - 4)² = 25² - 24²

(R - 4)² = 625 - 576  = 49

(R - 4) = √49 = 7

R = 7 + 4 = 11 cm

Panjang jari-jari Lingkaran lainnya adalah 11 cm (C)

Nomor 28 ::

Panjang tangga = sisi miring = c = 2,5 m

Jarak ujung bawah tangga dg tembok = sisi alas = a = 1,5 m

Tinggi ujung atas tangga dari lantai = b = ......

Berlaku teorema phytagoras :

c² = a² + b²

b² = c² - a²

b² = 2,5² - 1,5²

b² = 6,25 - 2,25

b² = 4

b = √4

b = 2 m

Tinggi ujung atas tangga dari lantai adalah 2 meter (B)

Diketahui:

P = { 2, 3, 5, 7, 10, 12}

Q = { 1, 3, 5, 7, 9}

Hasil P − Q adalah ….

A. { 3, 5, 7}

B. { 2, 3, 10}

C. { 2, 10, 12}

D. { 2, 9, 10, 12}

Pembahasan:

Hasil P − Q adalah anggota P yang tidak sama dengan anggota Q.

Jadi, hasil P − Q adalah P − Q = {2, 10, 12}

29. jawaban soal matematika halaman 9 kelas 6​


1. Posisi titik A adalah pada bilangan - 7. Jika di kiri nol, maka A menjadi negatif.

2. Posisi titik B adalah pada bilangan 0. Dari - 2 maka ditambah dua menjadi B.

3. Posisi bendera adalah pada bilangan - 12. Bendera di sebelah kiri maka menjadi negatif.

4. Posisi Dina adalah pada bilangan 2. Posisi Dina berubah dari nol ditambah 20 lalu dikurangi 18.

5. Posisi Dika adalah pada bilangan - 1. Posisi Dika berubah dari nol dikurangi 8 lalu ditambah 7.

BILANGAN BULAT

Bilangan bulat terdiri atas tiga kelompok bilangan, yaitu

Bilangan bulat negatif
Terletak di sebelah kiri angka nol Bilangan nol Bilangan bulat positif
Terletak di sebelah kanan angka nol

Dari sebuah bilangan jika

Maju atau ke kanan sebanyak x bilangan, maka bilangan awal ditambahkan dengan x. Mundur atau ke kiri sebanyak x bilangan, maka bilangan awal dikurangi dengan x.

Penjelasan dengan langkah-langkah:

Diketahui:

Titik A berada 7 satuan di kiri nol Titik B berada 2 satuan di kanan - 2 Bendera berada 12 satuan di kiri nol Dina di titik nol
Maju 20 satuan
Mundur 18 satuan Dika di titik nol
Mundur 8 satuan
Maju 7 satuan

Ditanyakan:

A? B? Bendera? Posisi Dina? Posisi Dika?

Jawaban:

1. A = [tex]0 \:-\: 7[/tex]

A = - 7

2. B = [tex]- 2 \:+\: 2[/tex]

B = 0

3. Bendera = [tex]0 \:-\: 12[/tex]

Bendera = - 12

4. Dina = [tex]0 \:+\: 20 \:-\: 18[/tex]

Dina = [tex]20 \:-\: 18[/tex]

Dina = 2

5. Dika = [tex]0 \:-\: 8 \:+\: 7[/tex]

Dika = [tex]- 8 \:+\: 7[/tex]

Dika = - 1
