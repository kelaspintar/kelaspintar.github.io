Kelas B terdiri dari 28 siswa, rata-rata nila matematikanya adalah 7,0. Kelas C terdiri dari 32 siswa, rata-rata nila matematikanya adalah 6,6. Kelas D terdiri dari N siswa, rata-rata nila matematikanya adalah 7,4. Jika rata-rata nilai matematika gabungan keempat kelas tersebut adalah 6,82, tentuan banyaknya siswa Kelas D tersebut


Rata- rata total = Jumlah nilai seluruh kelas ÷ Banyak seluruh siswa
6,82 = (6,5 × 30 + 7 × 28 + 6,6 × 32 + 7,4 × N) ÷ (30 + 28 + 32 + N)
6,82 = (195 + 196 + 211,2 + 7,4N) ÷ (90 + N)
6,82 × (90 + N) = 602,2 + 7,4N
613,8 + 6,82N = 602,2 + 7,4N
7,4N - 6,82N = 613,8 - 602,2
0,58N = 11,6
N = 11,6 ÷ 0,58
N = 20

Banyak siswa kelas D adalah 20 siswa.

39. Nilai rata rata ulangan matematika 29 anak sd sukarajin adalah 72. Jika pada data ini ditambahkan nilai fakhri, ternyata rata ratanya menjadi 72,5 berapakah nilai fakhri?


Nilai Fakhri adalah 87.

Nilai rata - rata (dilambangkan dengan x̄) pada suatu kelompok adalah suatu bilangan yang mewakili keseluruhan data pada kelompok tersebut yang didapatkan dari jumlah nilai data dibagi dengan banyaknya data tersebut. Biasanya nilai rata - rata digunakan untuk mengevaluasi data atau membandingkan kelompok data yang satu dengan yang lainnya sehingga lebih cepat dan menggambarkan seluruh data namun tidak dapat digunakan untuk menentukan nilai data tertentu di antara sekelompok data. Rata - rata suatu data ditentukan dengan rumus : x̄ = [tex] \frac{jumlah \: data}{banyaknya \: data} [/tex].

Agar lebih jelas dalam penerapannya, simak pembahasan soal berikut.

PEMBAHASAN :

Nilai rata - rata ulangan matematika 29 anak SD Sukarajin adalah 72.

Pertama, kita hitung jumlah nilai ulangan 29 anak tersebut melalui rata - ratanya.

x̄ = [tex] \frac{jumlah \: data}{banyaknya \: data} [/tex]

72 = [tex] \frac{jumlah \: nilai \: ulangan}{29} [/tex]

Jumlah nilai ulangan = 72 × 29

Jumlah nilai ulangan = 2.088

Jika pada data ini ditambahkan nilai Fakhri, ternyata rata - ratanya menjadi 72,5. Berapakah nilai Fakhri?

Setelah Fakhri ikut susulan, berarti jumlah siswa yang ikut ulangan menjadi 30 anak, sedangkan jumlah nilai siswa sebelumnya akan ditambah x. Mengapa x? Karena nilai Fakhri belum diketahui dan akan dicari.

Maka,

72,5 = [tex] \frac{jumlah \: nilai \: ulangan + x}{30} [/tex]

Jumlah nilai + x = 72,5 × 30

2.088 + x = 2.175

x = 2.175 - 2.088

x = nilai Fakhri = 87
