Di Soal Diatas, Berupa Soal Materi Himpunan, Apa Itu Himpunan? Himpunan Adalah Materi Matematika Yang menjelaskan tentang Kumpulan Kumpulan Anggota atau Objek yang Sudah Jelas, Himpunan Dapat disajikan dengan 3 Cara yaitu :

• Dapat dinyatakan dengan Sifat Kata Kata (Sifat Keanggotaan)

- Misal :

B = {Bilangan Prima Kurang dari 10}

• Dapat dinyatakan dengan Mendaftarkan Anggota Himpunan Tersebut

Pendaftaran anggota

- Misal :

B = {2,3,5,7}

• Dapat dinyatakan dengan Notasi Pembentuk Himpunan {x | x}

↓

Biasanya Notasi Pembentuk Himpunan terdapat Tanda Banding yaitu :

< = Kurang dari

> = Lebih dari

≤ = Kurang dari sama dengan

≥ = Lebih dari sama dengan

B = {x | x < 10, x € Bilangan Prima}

Bunga = Objek

Warna Merah = Objek Penjelasan

Operasi Himpunan

Berupa Operasi Himpunan Yaitu :

• Operasi Uner : Komplemen

• Operasi Biner : Irisan, Gabungan, Selisih

Notasi :

Irisan = A n B

Gabungan = A U B

Komplemen = A^C

Selisih = A - B

Note : ^ = Pangkat

• A n B = Irisan Himpunan Yang Berarti Anggota A Dan Juga Anggota B

• A U B = Gabungan Himpunan Yang Berarti Anggota A atau Anggota B (Secara Gabung)

• A - B = Selisih Himpunan Yang Berarti Anggota A Namun Bukan Anggota B

Menggunakan metode faktorisasi prima:

105 = 3 * 5 * 7

405 = 3 * 3 * 3 * 3 * 5

Kita dapat melihat bahwa faktor bersama terbesar dari 105 dan 405 adalah 3 * 5 = 15.

Kita dapat membagi pembilang dan penyebut dengan 15:

(105 / 405) / 15 = (7 * 3 * 5) / (3 * 3 * 3 * 3 * 5)

Menghilangkan faktor bersama dari pembilang dan penyebut, kita dapat menyederhanakan pecahan menjadi:

7 / (3 * 3 * 3) = 7 / 27

Jadi, bentuk paling sederhana dari pecahan 105/405 adalah 7/27.
