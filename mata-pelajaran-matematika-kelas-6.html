Diketahui :

jumlah seluruh siswa = 32 siswa

menyukai matematika = 22 siswa

menyukai musik = 21 siswa

tidak menyukai keduanya = 4 siswa

Ditanyakan :

siswa yang hanya menyukai matematika saja adalah ?

penyelsaian =

( 22 - x ) + x + ( 21 - x ) + 4 = 32

22 + 21 + 4 - x = 32

47 - 32 = x

x = 15

15 siswa menyukai keduanya

siswa hanya suka matematika =

22 - 15 = 7 siswa

29. tlng bantu nnti kumpulmata pelajaran matematika, SD kelas 6pakai caraterimakasih ​


Jawaban:

bangun lingkaran

rumus keliling 2xπr

= 2×22/7×14

= 616/7

= 88 ( keliling utuh lingkaran )

= 88/2

= 44 ( keliling setengah lingkaran )

π = 22/7 atau 3,14 (jika kelipatan 7 maka menggunakan 22/7 jika bukan kelipatan 7 maka menggunakan 3,14)

bangun persegi panjang

r = jari jari

rumus keliling 2×(p+l)

= 2×(31+17)

= 2×(48)

= 96

p = panjang

l = lebar

jadi keliling bangun diatas adalah

keliling lingkaran ditambah keliling persegi panjang

= 44 + 96

= 140

"jadi keliling bangun diatas adalah 140cm²"
