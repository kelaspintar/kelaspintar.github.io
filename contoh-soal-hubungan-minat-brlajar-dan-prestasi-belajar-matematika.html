11. Tolong di bantu soal matematika minat​


Jawaban

x = -4

Pembahasan

Diketahui :

[tex]f(x) = ^{6} log(2x + 8) [/tex]

Ditanya :

asimtot tegak : ..... ???

Jawab :

[tex]f(x) = ^{6} log(2x + 8) [/tex]

cara mencari asimtot tegaknya adalah dengan menggunkan limit mendekati tak hingga ke fungsi f (x)

[tex] \frac{lim}{x - > \infty } x = ^{6} log(2x + 8) [/tex]

2x + 8 = 0

2x = -8

x = -4

maka asimtot tegaknya adalah

x = -4

21. Soal Matematika Minat Bab Vektor


Jawab:

cos θ = [tex]\frac{1}{6}\sqrt{3}[/tex]

Penjelasan dengan langkah-langkah:

a.c = 20

a(2a - b) = 20 ⇒  2a.a - a.b = 20

                           2|a|² - a.b = 20

                      2(2√3)² - a.b = 20

                              24 - a.b = 20

                      a.b = 24 - 20 = 4

cos θ = [tex]\frac{a.b}{|a||b|}=\frac{4}{2\sqrt{3}(4) } = \frac{1}{2\sqrt{3} } = \frac{1}{6}\sqrt{3}[/tex]
